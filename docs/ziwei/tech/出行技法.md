# 出行实战技法

## 核心技法

### 一、命迁对比法

判断适合留守还是外出发展的核心方法：

```
命宫 vs 迁移宫 强弱对比：

命强迁弱 → 宜本地发展，守住根基
命弱迁强 → 宜外地发展，出外有转机
命迁双强 → 内外皆通，灵活选择
命迁双弱 → 内外都难，需寻找突破点

权重分配：本宫50% + 对宫30% + 三合夹宫20%
```

### 二、出行类型分流法

不同出行目的关注不同宫位：

```
问：出行目的是什么？
├── 工作调动/求职 → 主看迁移+官禄
│   → 迁移化禄+官禄吉 = 外出求职好时机
├── 旅游/休闲 → 主看迁移+福德
│   → 迁移化禄+福德吉 = 旅游愉快
├── 移民/长期定居 → 主看迁移+田宅
│   → 迁移化禄+田宅吉 = 定居顺利
├── 留学 → 主看迁移+官禄+父母
│   → 迁移吉+官禄吉+父母吉 = 留学成功
└── 商务出差 → 主看迁移+财帛
    → 迁移化禄+财帛吉 = 出差有利
```

### 三、出行安全三重检查法

```
第一重：原局迁移宫定性
  🟢旺 → 出行安全基本有保障
  🟡中 → 出行需注意常规安全
  🔴凶 → 先天出行风险高

第二重：大限出行安全
  大限化忌入迁移 × 原局🟢旺 → 有阻碍但能度过
  大限化忌入迁移 × 原局🔴凶 → 十年出行风险高

第三重：流年出行安全
  流年化忌入迁移+煞 = 该年出行需特别注意
  流年化禄入迁移 = 该年出行安全顺利
```

### 四、三层时间维度分析

#### 原局出行底盘
- 迁移宫V3.0定性 → 出行承载力
- 🟢实旺 → 出行运极佳，外有贵人
- 🟡磨炼 → 出行运中等，需做好准备
- 🟠虚浮 → 出行虚化，看似好实际差
- 🔴凶危 → 出行风险高，保守为上

#### 大限出行走向
```
大限走迁移宫 → 十年出行运被引动
大限化禄入迁移 × 🟢旺 → 十年外出发展好
大限化禄入迁移 × 🟡中 → 有推动需主动
大限化忌入迁移 × 🟠弱 → 出行受阻
```

#### 流年出行运
```
流年四化 × 大限方向 × 原局定性 = 今年出行表现

吉 × 吉 × 🟢旺 = 最佳出行年
吉 × 凶 × 🟡中 = 出行有波折但有收获
凶 × 凶 × 🔴凶 = 出行高风险年
```

## 实战断语

### 迁移宫主星断语

#### 紫微在迁移宫
- 旺宫：出外有贵人，受人尊重，适合在外发展
- 平宫：出外有人脉，但需主动社交
- 陷宫：出外表面风光实际空虚

#### 天机在迁移宫
- 旺宫：出外灵活机动，适合短期出行
- 平宫：出外多变，需做好计划
- 陷宫：出外焦虑不安，需有人陪同

#### 太阳在迁移宫
- 旺宫：出外光明正大，适合公众场合
- 平宫：出外需注意热情是否持久
- 陷宫：出外遇阻碍，不宜高调

#### 贪狼在迁移宫
- 旺宫：出外多社交机会，人缘好
- 平宫：出外桃花多，注意分寸
- 陷宫：出外放纵，需自律

### 四化入迁移断语

| 四化 | 断语 |
|------|------|
| 化禄入迁移 | 出行顺利，外地有财禄 |
| 化权入迁移 | 在外地有主导权，适合争取 |
| 化科入迁移 | 因外出得名声，遇贵人 |
| 化忌入迁移 | 出行不顺，注意安全 |

### 出行安全预警

| 信号 | 预警等级 | 断语 |
|------|---------|------|
| 流年化忌入迁移+擎羊 | 🔴高危 | 今年出行安全风险，需特别注意 |
| 大限化忌入迁移 | 🟠中危 | 十年出行不顺期 |
| 迁移宫煞聚 | 🟡持续 | 先天出行需注意安全 |
| 迁移宫化忌+天马+凶聚 | 🔴高危 | 交通出行需特别警惕 |

## 特殊情况处理

### 迁移宫空宫
- 借对宫（命宫）星曜
- 出行方向不确定
- 需看行运引动

### 异地发展判断
```
综合评估是否适合异地发展：

有利因素（每项+1）：
- 迁移宫旺于命宫
- 迁移宫有天马或禄存
- 迁移宫有吉星（天魁天钺左右昌曲）
- 大限走迁移宫+化禄

不利因素（每项-1）：
- 命宫旺于迁移宫
- 迁移宫化忌+煞
- 田宅宫极好（根扎得深）
- 家庭责任重

总分 > 0 → 适合外地发展
总分 ≤ 0 → 宜留守本地
```

### 交通安全特别检查
- 迁移宫擎羊+蜚廉+白虎+破碎+阴煞+地劫+化忌 → 车祸风险
- 再会天机+陀罗+天马+巨门 → 风险加倍
- 出行前必须提示安全注意
