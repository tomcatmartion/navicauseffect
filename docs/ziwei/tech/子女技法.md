# 子女实战技法

## 核心技法

```
Step 1: 子女宫 → 子女的基本特质和缘分深浅
Step 2: 田宅宫 → 家庭环境是否支持子女成长
Step 3: 命宫+夫妻宫 → 父母自身条件和婚姻环境
```

### 三、三层时间维度分析

#### 原局子女缘
- 子女宫V3.0定性 → 子女缘分的承载力
- 🟢实旺 → 子女缘极佳，子女优秀
- 🟡磨炼 → 子女缘中等，需用心养育
- 🟠虚浮 → 子女缘虚化，表面有实际付出多
- 🔴凶危 → 子女缘薄，养育辛苦

#### 大限子女运
```

大限化禄入子女宫 × 🟢旺 → 子女优秀期
```

```
流年红鸾入子女宫 → 可能有生育机会
```

## 特殊情况处理

### 子女宫空宫
- 借对宫星曜，子女缘不确定
- 需看行运引动
- 不代表无子女，只是缘分来得晚或不明显

### 子女宫逢空劫
- 子女缘薄或聚少离多
- 不一定无子女，可能子女在外地发展
- 养育过程中付出多但回报不明显

### 生育困难信号
- 子女宫化忌+煞聚 → 生育不顺
- 子女宫空劫+大耗 → 子女少或缘薄
- 田宅宫凶 → 家庭环境不利于生育
