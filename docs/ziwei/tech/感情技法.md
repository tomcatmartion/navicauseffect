# 感情实战技法

## 核心技法

### 一、感情三线诊断法

```
命迁线（缘分基础）：命宫+迁移宫 → 你是什么样的人+社交缘
夫官线（感情实质）：夫妻宫+官禄宫 → 感情质量+事业对感情的影响
财福线（感情品质）：财帛宫+福德宫 → 物质基础+精神满足
```

### 二、配偶画像技法

从夫妻宫主星推断配偶特征：

| 主星 | 配偶特质 | 相处模式 |
|------|---------|---------|
| 紫微 | 地位高、有主见、要求高 | 尊重但不卑微，需适当顺从 |
| 天机 | 聪明灵活、多变 | 需给空间，接受变化 |
| 太阳（旺） | 阳光开朗、热情 | 直接表达，积极互动 |
| 武曲 | 刚强独立、重事业 | 独立相处，不粘人 |
| 天同 | 温柔浪漫、享受生活 | 轻松相处，共同享受 |
| 廉贞 | 有原则、感情纠葛 | 需明确界限，避免暧昧 |
| 天府 | 稳重务实、善理财 | 稳定相处，家庭为重 |
| 太阴（旺） | 温柔细腻、顾家 | 细心呵护，注重情感 |
| 贪狼 | 多才多艺、桃花重 | 需管控社交，增加共同活动 |
| 巨门 | 善辩、多疑 | 坦诚沟通，避免猜忌 |
| 天相 | 温和体贴、随和 | 需给方向，帮助决策 |
| 天梁 | 正直善良、年长可能 | 尊重长辈式关怀 |
| 七杀 | 独立强势、个性鲜明 | 需要独立空间，直接沟通 |
| 破军 | 变化大、不安分 | 需要新鲜感，保持变化 |

### 三、感情阶段判断法

不同阶段关注不同宫位：

```
阶段一：寻找对象
  → 重点看迁移宫（社交缘）+ 夫妻宫（缘分有无）
  → 桃花星是否被引动

阶段二：恋爱中
  → 重点看夫妻宫（感情质量）+ 福德宫（感受）
  → 是否有煞忌冲克

阶段三：考虑结婚
  → 重点看夫妻宫+命宫（双方条件匹配度）
  → 大限是否走夫妻宫运

阶段四：婚后经营
  → 重点看田宅宫（家庭基础）+ 财福线（物质保障）
  → 流年化忌冲夫妻宫时需注意
```

### 四、三层时间维度分析

#### 原局感情底盘
- 夫妻宫V3.0定性 → 感情承载力
- 🟢实旺 → 感情运极佳，婚姻美满
- 🟡磨炼 → 感情中等，需经营
- 🟠虚浮 → 感情虚化，桃花不实
- 🔴凶危 → 感情多波折，需谨慎

#### 大限感情走向
```
大限宫干四化 × 原夫妻宫定性 = 大限感情效果

大限化禄入夫妻 × 🟢旺 → 十年感情甜蜜
大限化禄入夫妻 × 🟡中 → 感情有推动，需主动
大限化忌入夫妻 × 🟠弱 → 感情受创，需忍耐
```

#### 流年感情转折
```
流年四化 × 大限方向 × 原局定性 = 今年感情表现

吉 × 吉 × 🟢旺 = 最佳恋爱/结婚年
吉 × 凶 × 🟡中 = 有缘分但有阻碍
凶 × 凶 × 🔴凶 = 感情危机年
```

### 五、桃花判断技法

#### 正桃花（好桃花）
- 夫妻宫桃花星 + 正气吉星 → 自由恋爱，感情好
- 流年红鸾/天喜入命/夫妻宫 → 正缘年
- 大限走夫妻宫+化禄 → 十年正桃花

#### 偏桃花（纠缠桃花）
- 夫妻宫桃花星 + 煞星 → 感情纠纷
- 贪狼化忌 → 欲望失控
- 廉贞加煞 → 感情纠葛

#### 桃花时间窗口
- 红鸾/天喜入命宫或夫妻宫 → 桃花年
- 流年化禄入夫妻宫 → 缘分年
- 大限走夫妻宫 → 感情引动十年

## 实战断语

### 夫妻宫主星断语

#### 紫微在夫妻宫
- 旺宫：配偶有地位有能力，但要求高，需尊重对方
- 平宫：配偶有主见，相处需磨合
- 陷宫：配偶控制欲强但无实力，关系紧张

#### 贪狼在夫妻宫
- 旺宫+吉：配偶多才多艺，感情丰富
- 平宫：桃花多但感情不稳定
- 陷宫+煞：感情纠葛，外遇风险

#### 七杀在夫妻宫
- 旺宫：配偶独立强势，相处直接
- 平宫：冲突多，需大量磨合
- 陷宫：刑克重，宜晚婚

### 四化入夫妻断语

| 四化 | 断语 |
|------|------|
| 化禄入夫妻 | 感情甜蜜期，适合确定关系 |
| 化权入夫妻 | 感情中争夺主导权，激烈但热烈 |
| 化科入夫妻 | 因学识/名声结缘，感情优雅 |
| 化忌入夫妻 | 感情受挫期，需冷静处理 |

### 婚姻危机预警

| 信号 | 预警等级 | 断语 |
|------|---------|------|
| 流年化忌入夫妻+煞 | 🔴高危 | 今年感情危机，需特别经营 |
| 大限化忌入夫妻 | 🟠中危 | 十年感情波折期 |
| 命化忌冲夫妻 | 🟡持续 | 自身性格影响感情 |
| 夫妻宫六煞聚会 | 🔴高危 | 刑克极重，需晚婚化解 |

## 特殊情况处理

### 夫妻宫空宫
- 借对宫星曜，感情方向不确定
- 晚婚为宜
- 行运引动时才有明确感情

### 离婚信号综合判断
需同时满足以下至少2个条件才可提及离婚风险：
1. 夫妻宫逢四煞之一
2. 夫妻宫化忌加煞
3. 命宫失陷+夫妻宫失陷
4. 大限+流年双重化忌入夫妻宫
5. 六煞聚会在夫妻宫或三方

### 复合/复合判断
- 流年化禄入夫妻宫 → 有复合机会
- 但如果原局夫妻宫煞重 → 复合后仍波折
- 化忌后接着化禄 → 关系修复期
