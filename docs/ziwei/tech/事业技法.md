# 事业实战技法

## 核心技法

### 一、事业五步诊断法

```
Step 1: 命宫定基调 → 事业上限和性格基调
Step 2: 官禄宫看职业 → 具体职业方向
Step 3: 迁移宫看机遇 → 外在贵人和发展空间
Step 4: 财帛宫看收入 → 事业与财运的关系
Step 5: 综合评分 → 事业整体格局判断
```

### 三、事业强弱三线判断法

以命迁线、夫官线、财福线三条线综合判断：

```
命迁线（事业根基）：命宫+迁移宫 → 事业的基础和外部机遇
夫官线（事业发展）：夫妻宫+官禄宫 → 事业的具体表现
财福线（事业收益）：财帛宫+福德宫 → 事业带来的收入和满足感

最强之宫为主，其余为辅
```

### 四、三层时间维度分析

#### 原局事业底盘
- 官禄宫V3.0定性 → 事业承载力
- 🟢实旺 → 事业承载力极强，可向高处发展
- 🟡磨炼 → 能量中性，需积累和等待
- 🟠虚浮 → 事业虚有其表，不宜高调
- 🔴凶危 → 事业多阻碍，保守为上

#### 大限事业走向
```
大限宫干四化 × 原官禄宫定性 = 大限事业效果

大限化禄入官禄 × 🟢旺 → 十年事业大旺，升迁机会多
大限化禄入官禄 × 🟡中 → 事业有推动，需主动把握
大限化忌入官禄 × 🟠弱 → 事业受挫，宜守不宜攻
大限化忌入官禄 × 🟢旺 → 暂时困难，能扛过去
```

#### 流年事业转折
```
流年四化 × 大限方向 × 原局定性 = 今年事业表现

吉 × 吉 × 🟢旺 = 最佳晋升年
吉 × 凶 × 🟡中 = 有转机但有阻力
凶 × 凶 × 🟠弱 = 事业危机年
```

## 实战断语

### 官禄宫主星断语

#### 紫微在官禄宫
- 旺宫：领导力强，事业掌控力高，适合管理岗
- 平宫：有管理意识但执行力需提升
- 陷宫：独断无力，事业受阻

#### 天机在官禄宫
- 旺宫：思维敏捷，适合技术策划类，发展灵活
- 平宫：多虑易变，需选定方向
- 陷宫：焦虑多变，事业不稳定

#### 太阳在官禄宫
- 旺宫：事业积极向上，容易得贵人认可
- 平宫：热情中等，有虎头蛇尾倾向
- 陷宫：盲目无方向，事业停滞

#### 七杀在官禄宫
- 旺宫：事业冲劲足，适合竞争性行业
- 平宫：冲动与理性并存，需指导
- 陷宫：莽撞破坏，事业多波折

### 四化入官禄断语

| 四化 | 断语 |
|------|------|
| 化禄入官禄 | 事业顺遂年，适合深耕或扩大 |
| 化权入官禄 | 掌权年，适合争取管理岗或创业 |
| 化科入官禄 | 专业认可年，适合考证或评职称 |
| 化忌入官禄 | 事业受挫年，注意职场人际 |

### 职业转型信号

| 信号 | 含义 |
|------|------|
| 官禄宫杀破狼 | 行业变化大，转型机会多 |
| 大限化忌入官禄 | 不宜从商，宜上班 |
| 流年化忌入官禄+迁移化禄 | 换工作反而是好机会 |
| 财帛宫干飞化忌入某宫 | 该宫主星代表行业可能转型 |

## 特殊情况处理

### 官禄宫空宫
- 借对宫星曜（事业方向不确定）
- 需看命宫主星来定大方向
- 行运走到时才有明确事业方向

### 创业 vs 上班判断
```
问：适合创业还是上班？

创业条件：
- 命宫+官禄宫都旺
- 迁移宫有贵人星（天魁天钺）
- 官禄宫有化权或化禄
- 大限走财运或官禄运

上班条件：
- 官禄宫稳定星（紫微、天府、天相）
- 无杀破狼的冲动
- 大限化忌入财帛/迁移（不宜冒险）
```

### 升职时机判断
- 大限化权入官禄 → 这十年有晋升机会
- 流年化禄或化权入官禄 → 今年是升职窗口
- 命宫旺 + 官禄宫旺 + 行运吉 → 升职确定性强
