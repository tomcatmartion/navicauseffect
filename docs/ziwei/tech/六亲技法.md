# 六亲实战技法

## 核心技法

### 一、六亲宫位定位法

紫微斗数中每个六亲关系对应特定宫位：

```
父亲 → 父母宫 + 太阳
母亲 → 父母宫（次要） + 兄弟宫 + 太阴
配偶 → 夫妻宫 + 男看太阴/女看太阳
兄弟 → 兄弟宫 + 天机星
子女 → 子女宫 + 田宅宫
朋友/下属 → 奴仆宫
上司/靠山 → 父母宫 + 迁移宫
```

### 二、贵人识别技法

从各宫位判断贵人和小人：

#### 贵人星（出现在六亲宫位时）
- **天魁/天钺** → 贵人星，该六亲是命中贵人
- **左辅/右弼** → 助力星，该六亲有助益
- **化禄** → 该六亲带来财运助力
- **化科** → 该六亲带来名声助力

#### 小人识别
- **奴仆宫化忌+煞** → 命中小人宫
- **奴仆宫逢擎羊/陀罗** → 朋友中有人不诚实
- **流年化忌入奴仆** → 该年注意人际风险
- **大限走奴仆宫化忌** → 十年人际纠纷多

### 三、太岁入卦法（互动关系分析）

当需要分析与特定六亲的互动时：

```
Step 1: 获取对方出生年份（生肖/地支）
Step 2: 找到该地支在命盘中的宫位
Step 3: 该宫位就是对方的"入卦宫位"
Step 4: 分析该宫位的星曜旺衰 → 判断互动质量
```

**应用场景**：
- 分析与合伙人的关系
- 分析与特定家人的关系
- 分析与上司/下属的关系

### 四、三层时间维度分析

#### 原局六亲缘
- 各六亲宫位V3.0定性 → 六亲缘分承载力
- 🟢实旺 → 六亲缘好，关系和谐
- 🟡磨炼 → 六亲缘中等，需经营
- 🟠虚浮 → 六亲缘虚化，表面有实际疏远
- 🔴凶危 → 六亲缘薄，关系紧张

#### 大限六亲走向
```
大限走某六亲宫 → 十年该关系被引动
大限化禄入某六亲宫 × 🟢旺 → 十年关系好
大限化忌入某六亲宫 × 🟠弱 → 关系紧张
```

#### 流年六亲运
```
流年化禄入某六亲宫 → 今年关系融洽
流年化忌入某六亲宫 → 今年关系有矛盾
```

## 实战断语

### 各宫位主星断语

#### 紫微在六亲宫
- 旺宫：该六亲有地位有能力，是你的贵人
- 平宫：该六亲有主见，关系尚可
- 陷宫：该六亲要求高但实力不足，关系紧张

#### 天府在六亲宫
- 旺宫：该六亲稳重可靠，是家庭支柱
- 平宫：该六亲务实，关系稳定
- 陷宫：该六亲过于保守，需主动沟通

#### 七杀在六亲宫
- 旺宫：该六亲独立有主见，关系直接
- 平宫：该六亲叛逆，需包容
- 陷宫：该六亲极端，关系困难

### 四化入六亲断语

| 四化 | 断语 |
|------|------|
| 化禄入六亲宫 | 与该六亲关系好，得其助力 |
| 化权入六亲宫 | 该六亲强势，需适当退让 |
| 化科入六亲宫 | 该六亲有才华，因他得名声 |
| 化忌入六亲宫 | 与该六亲关系受阻，需多沟通 |

### 人际风险预警

| 信号 | 预警等级 | 断语 |
|------|---------|------|
| 流年化忌入奴仆+煞 | 🔴高危 | 今年人际风险高，注意被骗 |
| 大限化忌入奴仆 | 🟠中危 | 十年人际纠纷期 |
| 奴仆宫煞聚 | 🟡持续 | 人际关系复杂，择友需谨慎 |

## 特殊情况处理

### 六亲宫空宫
- 借对宫星曜
- 与该六亲关系不确定
- 行运引动时关系才有变化

### 多个六亲宫同时化忌
- 代表该阶段家庭关系整体紧张
- 需区分各宫位的轻重缓急
- 优先处理化忌+煞最重的宫位

### 合伙关系分析
- 兼看兄弟宫（合伙人关系）+ 奴仆宫（下属关系）
- 需获取合伙人生年 → 太岁入卦法
- 兄弟宫化忌 → 合伙需谨慎
