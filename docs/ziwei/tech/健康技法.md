# 健康实战技法

## 核心技法

### 一、疾病四步诊断法

```
Step 1: 看命宫定基调 → 体质强弱、生命力
Step 2: 看疾厄宫找方向 → 五行方向、疾病类型
Step 3: 看父母宫查遗传 → 先天体质、家族病史
Step 4: 综合判断定轻重 → 结合行运判断时间
```

### 二、五行病机分析法

根据疾厄宫主星的五行属性，推断疾病方向：

```
五行路径：
木（天机/贪狼）→ 肝胆 → 筋骨/神经/情志
火（太阳/廉贞）→ 心血管 → 血液/眼睛/炎症
土（紫微/天府/天同）→ 脾胃 → 消化/肌肉/口
金（武曲/太阴）→ 肺部 → 呼吸/皮肤/骨骼
水（天同/巨门/贪狼）→ 肾脏 → 泌尿/生殖/耳
```

**细化规则**：
- 主星旺宫 → 五行对应系统健康
- 主星平宫 → 五行对应系统需注意
- 主星陷宫 → 五行对应系统易出问题
- 加煞忌 → 疾病加重

### 三、疾病时间推断法

#### 原局体质判断
```
命宫+疾厄宫综合定性：

双宫 🟢旺 → 先天体质好，少病
一旺一平 → 体质中等，偶有小病
双宫 🟡中 → 体质一般，需养生
一平一弱 → 体质偏弱，需注意
双宫 🔴凶 → 先天体质差，需长期保健
```

#### 大限疾病引动
```
大限走疾厄宫 → 该十年健康被重点关注
大限化忌入疾厄 × 原疾宫🟢旺 → 有病但能治好
大限化忌入疾厄 × 原疾宫🔴凶 → 疾病加重
```

#### 流年疾病应期
```
流年化忌入疾厄 = 该年疾病触发点
流年凶星入疾厄 = 对应系统的急性发作

应期判断：
1. 找到原局疾厄宫的薄弱五行
2. 找到流年化忌落入的宫位
3. 两者交叉 = 疾病可能发作的时间和类型
```

### 四、体质分类技法

根据命宫+疾厄宫主星组合，判断体质类型：

| 体质类型 | 特征 | 养生方向 |
|---------|------|---------|
| 木型体质（天机/贪狼） | 神经敏感、肝气旺 | 疏肝理气、静心养神 |
| 火型体质（太阳/廉贞） | 精力旺盛、心火旺 | 降火清热、规律作息 |
| 土型体质（紫微/天府/天同） | 消化敏感、脾虚 | 健脾养胃、饮食规律 |
| 金型体质（武曲/太阴） | 呼吸敏感、肺弱 | 润肺养阴、防寒保暖 |
| 水型体质（巨门/天同） | 肾气偏弱、畏寒 | 补肾固本、适度运动 |

## 实战断语

### 疾厄宫主星断语

#### 紫微入疾厄
- 旺宫：脾胃功能好，但需注意饮食节制
- 平宫：消化系统中等，注意饮食规律
- 陷宫+煞：脾胃疾病风险，注意消化系统

#### 天机入疾厄
- 旺宫：神经系统灵敏，但不宜过度用脑
- 平宫：偶有神经紧张，需放松
- 陷宫+煞：神经系统疾病风险

#### 武曲入疾厄
- 旺宫：肺部强健，但需防金属性伤害
- 平宫：呼吸系统中等
- 陷宫+煞：肺部疾病风险，注意呼吸道

#### 廉贞入疾厄
- 旺宫：血液循环好
- 平宫：注意血液指标
- 陷宫+化忌：血液疾病风险

### 四化入疾厄断语

| 四化 | 断语 |
|------|------|
| 化禄入疾厄 | 身体好转期，适合做全面体检和调理 |
| 化权入疾厄 | 体质强壮但可能透支，注意劳逸结合 |
| 化科入疾厄 | 找到好的治疗方案，适合就医 |
| 化忌入疾厄 | 健康红灯期，对应系统需重点检查 |

### 伤灾预警断语

| 信号 | 预警等级 | 断语 |
|------|---------|------|
| 流年化忌入疾厄+擎羊 | 🔴高危 | 今年血光外伤风险，注意安全 |
| 大限化忌入疾厄+煞 | 🟠中危 | 十年健康注意期 |
| 疾厄宫煞聚 | 🟡持续 | 先天体质弱，需长期养生 |
| 迁移宫凶聚+天马 | 🔴高危 | 出行安全需特别注意 |

## 特殊情况处理

### 疾宫凶但命宫吉
- 不一定是大病
- 可能映射为事业或财运的压力
- 综合看财官宫的吉凶来区分

### 遗传疾病判断
- 父母宫不好 → 有遗传风险
- 结合疾厄宫五行 → 判断遗传方向
- 父母宫化忌冲疾厄 → 遗传疾病可能显现

### 心理疾病判断
- 福德宫化忌 → 精神压力大
- 福德宫化忌+煞 → 心理健康需关注
- 福德宫化忌冲命宫 → 精神问题影响整体状态
