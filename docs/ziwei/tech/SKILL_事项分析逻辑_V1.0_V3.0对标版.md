# SKILL：事项分析逻辑（V3.0对标版）

> **版本**: V1.0 · V3.0对标版  
> **类型**: 分析执行规范  
> **特色**: 原局+行运+流年三层分析，明确使用V3.0定性等级

---

## 📌 核心原则

### 分析基础：V3.0评分是第一步

```
Step 1: 用V3.0给12宫打分
        ↓
        获得：Final_Score（数值）+ 定性（🟢/🟡/🟠/🔴/⚫）
        ↓
Step 2: 根据定性查星曜赋性库
Step 3: 进行事项分析（用V3.0定性而非模糊描述）
```

### 三层时间维度（基于V3.0定性）

| 层级 | 定性基准 | 作用 |
|------|--------|------|
| **原局** | V3.0的Final_Score定性 | 定底盘 - 先天格局承载力 |
| **行运** | 大限宫干四化 × 原宫位V3.0定性 | 定激活 - 十年方向 |
| **流年** | 流年四化 × 大限方向 × 原宫位V3.0定性 | 定引动 - 今年触发点 |

## 第二阶段：原局底盘分析（V3.0框架）

### 步骤一：命宫全息底色带入

#### 数据来源

```
来源：SKILL_命主性格定性 + SKILL_宫位原生能级评估_V3.0

需要的信息：
1. 命宫的Final_Score和定性（🟢/🟡/🟠/🔴/⚫）
2. 命宫的主星和星曜组合
3. 命宫的制煞能力等级
```

#### 在事项分析中的应用

```
命宫定性是调节基准：

🟢实旺 → 命主能驾驭事项宫的凶象
🟡磨炼 → 命主对事项宫有限度支持  
🟠虚浮 → 命主无法驾驭事项宫的凶象
🔴凶危 → 命主本身难保，无力支持
⚫绝败 → 跳过分析
```

### 步骤二：识别原局关键格局（基于V3.0定性）

#### 宫位定性基准

**重要**：以下所有判断都基于V3.0的Final_Score定性

| V3.0定性 | 分数段 | 承载力 | 格局判断 |
|---------|--------|--------|--------|
| 🟢实旺 | 7.5～8.5 | 强 | 制煞为用可充分发挥，护佑有效 |
| 🟢磨炼 | 6.0～7.4 | 中强 | 整体强旺，护佑基本有效 |
| 🟡磨炼 | 4.5～5.9 | 中 | 能量中性，护佑一般有效 |
| 🟠虚浮 | 3.0～4.4 | 弱 | 虚华，护佑易破损 |
| 🔴凶危 | 1.5～2.9 | 极弱 | 逞凶，护佑无效 |
| ⚫绝败 | <1.5 | 无 | 无法分析 |

#### 空宫处理规则

| 情况 | 处理方法 | 说明 |
|------|--------|------|
| **事项宫位空宫** | 借对宫星曜+参考对宫V3.0定性 | 吉象减半，凶象更凶 |
| **空宫命宫** | 对宫投射为主+看对宫V3.0定性 | 对宫定性决定命主能级 |
| **空宫本身** | 不等于无结果，看承载力 | 空宫V3.0定性较弱，需外力推动 |

#### 格局保护机制（由V3.0定性确定成立）

**以下判断都需要该宫位V3.0定性≥🟡磨炼才能有效**：

| 格局名称 | 构成条件 | 定性要求 | 作用 |
|---------|--------|--------|------|
| **双禄夹宫** | 左右各有禄星 | 本宫≥🟡 | 护佑有效，凶限可保底 |
| **禄存在宫** | 禄存落在本宫 | 本宫≥🟡 | 基本盘保底 |
| **天府守财库** | 天府在财库宫 | 天府≥🟡 | 财库有制煞能力 |
| **紫微+天府** | 北斗双星同宫 | 双星≥🟡 | 制煞能力极强 |

**⚠️ 注意**：
```
如果本宫V3.0定性为🟠虚浮或以下
这些格局即使成立，也是虚化的，无法真正护佑
需标注：「格局虚华」
```

#### 关键大格局识别表

| 格局名称 | 成格条件 | V3.0定性要求 | 效力判断 |
|---------|--------|-----------|--------|
| **府相朝元** | 命对武贪+三合天府天相 | 三方≥🟢磨炼 | 强大格局 |

| **阳梁昌禄** | 太阳+天梁+昌+禄 | 四星≥🟡磨炼 | 科甲名声 |

### 步骤三：事项宫位四维合参（V3.0框架）

#### 目标

根据V3.0的定性，综合判断事项宫位的承载能力

#### 四维合参流程

**输入：各宫位的V3.0 Final_Score + 定性**

| 维度 | 判断方法 | V3.0应用 |
|------|--------|--------|
| **本宫** | 直接使用V3.0定性 | 该宫的Final_Score决定承载力 |
| **对宫** | 查对宫的V3.0定性 | 对宫强则加强，对宫弱则制约 |
| **合宫** | 综合两个三合宫的V3.0定性 | 合宫强则支撑，合宫弱则受压 |
| **临宫** | 综合两个邻宫的V3.0定性 | 临宫定性决定夹制效果 |

#### 具体判断要点

**本宫定性**（V3.0直接给出）：
```
🟢实旺（7.5-8.5）→ 承载力极强，可充分发挥
🟢磨炼（6.0-7.4）→ 承载力中强，有小幅摩擦
🟡磨炼（4.5-5.9）→ 承载力中等，能量平衡
🟠虚浮（3.0-4.4）→ 承载力弱，虚华需警惕
🔴凶危（1.5-2.9）→ 承载力极弱，无底线
⚫绝败        → 跳过分析
```

**对宫投射判断**：
```
对宫定性 ≥ 本宫定性 1级以上
  → 对宫有力加强本宫

对宫定性 < 本宫定性 1级以上
  → 对宫有力制约本宫

对宫定性 与本宫相当
  → 对宫中立影响
```

**三合支撑判断**：
```
两个三合都 ≥🟢磨炼
  → 强力支撑，原局有利

其中一个 ≥🟡磨炼，一个 <🟡
  → 支撑不足，侧翼受压

两个都 <🟡磨炼
  → 支撑虚弱，无后援
```

**临宫夹制判断**：
```
两个邻宫定性相当 且都≥🟢磨炼
  → 均衡夹制，激励效果

两个邻宫定性差异大
  → 失衡夹制，压制或松散

一个邻宫≥🟢磨炼，一个<🟠虚浮
  → 完全失衡，一强一弱
```

**命宫调节基准**：
```
命宫 ≥🟢磨炼
  → 能驾驭事项宫的凶象，制煞有效

命宫 为🟡磨炼
  → 对事项宫有限度支持

命宫 <🟡磨炼
  → 无力驾驭事项宫凶象，直接穿透
```

#### 最终输出：原局底盘定性（V3.0定性等级）

**示例模板**：
```
【原局底盘定性】

事项宫位：财帛宫
V3.0定性：Final_Score 5.3，🟡磨炼（4.5-5.9）

本宫定性：🟡磨炼
  说明：中等承载力，能量平衡

对宫投射：强化
  对宫为🟢磨炼（6.8），对宫强于本宫 → 有力加强

三合支撑：基本完整
  一个为🟢磨炼，一个为🟡磨炼 → 整体有利

临宫夹制：均衡激励
  两邻宫都为🟡磨炼 → 相当压力，激励有限

命宫调节：有效支持
  命宫为🟢磨炼 → 能驾驭本宫凶象

综合定性：🟡磨炼 偏强
  （本身中等+对宫加强+三合支持+命宫支持 → 偏向积极）

格局保护机制：基本完整
  双禄虽不夹此宫，但对宫禄存有支撑，护佑基本有效

特殊标注：
  • 此宫虽为磨炼，但周围支持力量较强，突破空间在大限吉化时显现
  • 大限凶化时需小心，但不会彻底崩盘（命宫支持+护佑完整）
```

## 第三阶段：行运分析（V3.0定性 × 大限四化）

### 核心逻辑

```
大限宫干四化 × 原宫位V3.0定性 = 大限效果

吉化 × 🟢强 → 吉象充分激活
吉化 × 🟡中 → 吉象需推动才能显现
吉化 × 🟠弱 → 吉象虚化为诱饵
忌化 × 🟠弱 → 凶象直接穿透
忌化 × 🟢强 → 凶象受制，相对轻微
```

## 第四阶段：流年引动（V3.0定性 × 流年四化 × 大限方向）

### 流年三层合参

```
流年四化 × 大限方向 × 原宫位V3.0定性 = 今年表现

吉 × 吉 × 🟢强 = 最佳推进窗口（力量叠加）
吉 × 吉 × 🟠弱 = 虚化救济（虽有帮助但打折）
忌 × 凶 × 🔴凶 = 风险最高（力量叠加）
忌 × 凶 × 🟡中 = 有风险但能相对承受（原宫支撑）
```

### 流年与大限方向判断

| 流年 | 大限 | 原宫V3.0 | 判断 | 建议 |
|------|------|--------|------|------|
| 吉 | 吉 | 🟢强 | ⬆️⬆️ 最佳推进窗口 | ✅ 主动出击 |
| 吉 | 吉 | 🟡中 | ⬆️↗️ 推进可能 | ✅ 适度推进 |
| 吉 | 吉 | 🟠弱 | ⬆️↔️ 虚化救济 | ⚠️ 谨慎推进 |
| 吉 | 凶 | 🟢强 | ↗️↔️ 转机但有阻 | ⚠️ 谨慎推进 |
| 凶 | 吉 | 🟡中 | ↔️⬇️ 波折不伤 | ⏸️ 维持为主 |
| 凶 | 凶 | 🟠弱 | ⬇️⬇️ 风险最高 | 🚫 预警规避 |
| 凶 | 凶 | 🔴凶 | ⬇️⬇️⬇️ 极度危险 | 🚫 必须规避 |
