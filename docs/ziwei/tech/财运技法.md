# 财运实战技法

## 核心技法

### 一、水库分析法

将财运系统比喻为水库，各宫位对应不同环节：

- **命身宫** = 天雨（天然助力、先天财运根基）
- **福德宫** = 源流（财源长短、理财智慧）
- **财帛宫** = 水渠（钱财运用、来财方式）
- **田宅宫** = 水库（财富积蓄、守财能力）
- **事业宫** = 水库管理员（通过事业管理财运）

**判断流程**：
1. 水渠（财帛）是否通畅 → 看财帛宫星曜旺衰
2. 源流（福德）是否充足 → 看福德宫财源星
3. 水库（田宅）是否漏水 → 看田宅宫有无空劫化忌
4. 管理员（事业）是否尽责 → 看官禄宫是否稳定

### 二、求财类型分流法

先确定求财类型，再选择对应宫位：

```
问：有无劳力付出？
├── 有劳力 → 问：有无场地？
│   ├── 无场地 → 主看财帛宫（打工/小规模自营）
│   ├── 有场地 → 问：规模大吗？
│   │   ├── 小 → 主看财帛宫
│   │   └── 大 → 兼看田宅宫
│   └── 涉及合伙？
│       ├── 自己当头 → 主看迁移宫
│       └── 出资不参与 → 主看福德宫
├── 无劳力（纯投资） → 主看福德宫，兼看迁移宫
└── 置产/收租 → 主看田宅宫
```

### 三、三层时间维度分析

#### 原局底盘（先天财运格局）
- 财帛宫V3.0定性 → 决定财运承载力
- 🟢实旺 → 承载力极强，可大胆经营
- 🟡磨炼 → 能量中性，需积累等待时机
- 🟠虚浮 → 吉星为诱饵，不宜冒险投资
- 🔴凶危 → 破财风险高，保守为上

#### 大限引动（十年财运方向）
```
大限宫干四化 × 原财帛宫定性 = 大限财运效果

大限化禄入财帛 × 🟢旺 → 十年财运大旺
大限化禄入财帛 × 🟡中 → 需推动才能显现
大限化忌入财帛 × 🟠弱 → 凶象穿透，破财
大限化忌入财帛 × 🟢旺 → 凶象受制，损失可控
```

#### 流年引动（具体年份财运）
```
流年四化 × 大限方向 × 原局定性 = 今年财运

吉 × 吉 × 🟢旺 = 最佳进财年（主动出击）
吉 × 吉 × 🟠虚 = 虚化救济（效果打折）
凶 × 凶 × 🔴凶 = 破财高风险年（必须规避）
```

## 实战断语

### 财帛宫主星断语

#### 武曲在财帛宫
- 旺宫：执行力强，忙中生财，适合金融投资、实业经营
- 平宫：需坚持积累，不可急进，有钻牛角倾向
- 陷宫：财务固执，破财风险大，不宜投机

#### 太阴在财帛宫
- 旺宫：理财有方，暗中生财，适合积蓄型投资
- 平宫：理财中等，需学习理财知识
- 陷宫：感情影响理财，易被欺骗

#### 紫微在财帛宫
- 旺宫：贵气生财，有贵人带财，适合高端行业
- 平宫：财运中上，需借助人脉
- 陷宫：虚有其表，面子消费大

#### 天府在财帛宫
- 旺宫：善守财，理财稳健，财库充实
- 平宫：守财能力中等
- 陷宫+空劫：守不住财，财库漏空

#### 贪狼在财帛宫
- 旺宫+火贪格：横发机遇，偏财运强
- 平宫：欲望驱使理财，起伏大
- 陷宫：贪心破财，不宜投机

### 四化入财帛断语

| 四化 | 断语 |
|------|------|
| 化禄入财帛 | 进财顺畅，适合扩大经营，财运年 |
| 化权入财帛 | 赚钱有魄力但易冒进，适合竞争性行业 |
| 化科入财帛 | 因名声得财，适合文教咨询类，稳定不暴 |
| 化忌入财帛 | 破财年，不宜投资投机，注意理财安全 |

### 破财预警断语

| 信号 | 预警等级 | 断语 |
|------|---------|------|
| 流年化忌入财帛+煞星 | 🔴高危 | 今年破财风险极大，务必保守 |
| 大限化忌入财帛 | 🟠中危 | 这十年财运受阻，不宜大投资 |
| 空劫夹财帛 | 🟡持续 | 天生财来财去，任何投资打五折 |
| 禄存独守+空劫 | 🔴高危 | 看似有财实则被劫 |

## 特殊情况处理

### 空宫财帛宫
- 借对宫所有星曜（打五折）
- 对宫好 → 财运有基础但不稳定
- 对宫差 → 财运无根基，靠行运带动

### 财福线（财帛-福德对宫关系）
- 财好福好 → 有财能享
- 财好福差 → 有钱但不快乐，或钱被精神消耗
- 财差福好 → 虽不富裕但知足安乐
- 财差福差 → 既穷且苦

### 合伙财运
- 合伙当头 → 主看迁移宫（领导力+人脉）
- 合伙出资不参与 → 主看福德宫（理财眼光）
- 必须参考合伙人生年 → 太岁入卦法判断互动
