# 财运解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 财帛宫（最核心）
- 正财来源、收入能力、钱财运用方式
- 财帛宫强且有财禄星 → 财源充足，经商有方
- 财帛宫弱或有空劫化忌 → 财来财去，不宜经商

### 2. 官禄宫（事业财）
- 事业带来的薪资收益、职业方向
- 有官就有财，事业的好坏直接影响财运
- 财帛宫强于官禄宫 → 宜从商
- 官禄宫强于财帛宫 → 宜公职或合伙

### 3. 福德宫（偏财/理财）
- 偏财运、理财眼光、现金流动向
- 纯投资（无劳力付出）→ 主看福德宫
- 福德宫也看财源长短：源远流长还是短暂爆发

### 4. 田宅宫（财库/不动产）
- 有无积蓄、能否守财、不动产运
- 财帛宫吉凶混杂时，可藉置业化解横发横破
- 田宅宫不好 → 财难留，难得好结果

### 5. 迁移宫（外财/人际财）
- 外地发展财运、人脉带来的财源
- 涉及外贸、出差、销售 → 兼看迁移宫
- 口才/业务/中介类收入 → 迁移宫权重增大

### 6. 命宫（财运根基）
- 命宫好坏决定财运上限
- 命宫主星平陷且财帛田宅宫均不好 → 财运基础薄弱

## 二、必须检查的星曜

### 正财星（主看）
- **武曲**：投资创业型财富，须经辛劳行动而获财，忙中生财
- **太阴**：积蓄积累型，暗中生财，重计划理财
- **天府**：善理财积蓄，但不一定善生财

### 偏财星
- **廉贞/贪狼**：须化禄或与禄存同宫，或成火贪/铃贪格，多意外或投机得财
- **七杀/破军/廉贞**：庙旺有禄有横财运
- **火贪格/铃贪格**：横发偏财格

### 辅助财星
- **禄存**：主食禄，主星入庙可聚财，加左右可不劳而获
- **化禄**：增财运，引发进财
- **化权**：增财力和竞争力
- **天马**：主动中取财，与禄存同宫为禄马交驰格

### 破财星曜（重点关注）
- **空劫**：财来财去，损财破财，财到手成空（财帛宫最忌）
- **化忌**：不宜经商投机
- **大耗**：与空劫同宫必大败
- **杀破狼机巨梁加煞**：好赌或破败

## 三、四化判断规则（严格遵守）

### 化禄入财帛宫
- 正财大旺，收入稳定增长
- 经商者客源广进
- 化禄落宫即该宫是财禄所在

### 化权入财帛宫
- 财运有竞争力，适合在财经领域掌权
- 赚钱有魄力，但可能过于激进

### 化科入财帛宫
- 因名声、才艺得财
- 财运稳定但不会暴富
- 适合文教、咨询类收入

### 化忌入财帛宫
- 财运受阻，当年容易破财
- 不宜经商投机，尤忌赌博
- 破财严重程度看有无煞星同宫

### 财帛宫自化忌
- 钱财留不住，赚了也存不下
- 需看田宅宫是否有化解

### 官禄宫化忌冲财帛宫
- 工作变动影响收入
- 事业不顺导致财运下跌

### 特殊四化组合
- **双禄交加**（本命或岁限有两个禄）→ 容易有横财和中奖
- **禄马交驰**（禄存与天马同对宫）→ 动态来财，越跑越赚
- **化忌冲田宅宫** → 财库漏财，不动产有损失

## 四、行运判断规则

### 大限财运
- 大限宫干化禄入财帛/福德/田宅 → 该十年财运好
- 大限宫干化忌入财帛 → 该十年财运受阻，注意理财
- 大限命宫落原局财帛宫 → 该十年财运被引动

### 流年财运
- 流年化禄入财帛宫 → 今年财运好
- 流年化忌入财帛宫 → 今年破财风险大
- 流年叠忌（两个化忌同时冲财） → 该年破财风险极大

### 三层合参逻辑
1. 先看原局财帛宫旺衰（定底盘）
2. 再看大限是否引动（定方向）
3. 最后看流年四化（定具体年份）
- 吉 × 吉 × 旺 = 最佳财运期
- 凶 × 凶 × 衰 = 破财高风险期

## 五、特殊格局判断

### 吉格（财运好）
- **紫府武曲居财帛+权禄** → 必为富翁
- **武曲太阴入庙守财宫无破耗** → 堆金积玉
- **日月夹财格** → 庙旺大富（忌羊陀空劫）
- **双禄交加** → 横财机遇
- **火贪/铃贪入财宫** → 横发机遇，无煞忌暴富
- **禄马交驰** → 动态生财，越忙越赚

### 凶格（财运差）
- **空劫入财帛** → 财来财去
- **化忌加煞入财帛** → 破财败财
- **劫空临财福格** → 贫贱格局
- **禄逢冲破格** → 有财难守
- **马落空亡格** → 动而无功

## 六、注意事项

### 易错点
- **空宫处理**：财帛宫空宫 → 借对宫星曜，吉象减半，凶象更凶
- **煞星双面性**：四煞入财宫庙旺时起护卫作用（增加竞争力），失陷时才是破财
- **偏财判断**：不是所有偏财都是好的，火贪格需无煞忌才真正暴富

### 禁止事项
- **禁止只看财帛宫**：必须结合官禄、福德、田宅综合判断
- **禁止给具体金额预测**：只能判断方向和量级感
- **禁止忽视空劫**：空劫夹财时所有财运判断打五折
- **禁止虚浮当旺论**：宫位V3.0定性为虚浮时，吉星为诱饵不可当真
- **禁止忽视命宫**：命宫是财运的上限，命弱则财强也难持久
