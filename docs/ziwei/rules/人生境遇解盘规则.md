# 人生境遇解盘规则（通用兜底）

> 当用户问题无法匹配具体领域时，使用本规则进行通用人生格局分析。

## 一、必须检查的宫位（按优先级）

### 1. 命宫（核心）
- 人生整体格局、性格基调、发展上限
- 命宫强弱决定整体运势走向

### 2. 迁移宫（外在发展）
- 社会活动能力、外在机遇
- 命宫主内，迁移宫主外

### 3. 官禄宫（事业方向）
- 职业倾向、事业成就

### 4. 财帛宫（财运基础）
- 经济状况、理财能力

### 5. 福德宫（精神层面）
- 精神状态、生活质量

### 6. 夫妻宫（感情基础）
- 感情状态、婚姻质量

## 二、分析框架

### 先看命宫定基调
- 命宫旺 → 先天条件好，整体运势有基础
- 命宫平 → 中等格局，靠行运推动
- 命宫弱 → 先天条件受限，需后天努力

### 再看六条主线
- **命迁线**：生命线、活动线
- **兄友线**：成就线、助力线
- **夫官线**：事业线、婚姻线
- **子田线**：桃花线、财库线
- **财福线**：金钱线、爱好线
- **父疾线**：头脑线、遗传线

### 三方四正综合
- 本宫星情 50%
- 对宫 30%
- 三合宫 20%

## 三、行运判断

### 大限看方向
- 大限走哪个宫位 → 十年重心在哪个领域
- 大限化禄入哪个宫 → 十年该领域有利
- 大限化忌入哪个宫 → 十年该领域受阻

### 流年看引动
- 流年化禄 → 该年顺遂
- 流年化忌 → 该年波折
- 流年与命盘本宫的互动 → 判断具体影响

## 四、输出原则

1. **先总后分**：先给整体格局判断，再分领域说明
2. **吉凶并陈**：优点和需要注意的方面都说
3. **具体不空泛**：结合命盘实际星曜说话，不用套话
4. **引导追问**：在分析中自然引出用户可能感兴趣的领域（事业、财运、感情等）

## 五、注意事项

- 无法确定具体领域时，给出人生整体格局的大判断
- 适当引导用户细化问题，以便做更精准的分析
- 不要面面俱到流水账，抓住最突出的 2-3 个重点
