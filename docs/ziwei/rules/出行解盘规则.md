# 出行解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 迁移宫（最核心）
- 社会活动空间、对外发挥能力
- 在外遭遇及人际关系
- 出行安全、外地发展运势
- 迁移宫强 → 适合出外发展，有贵人相助
- 迁移宫弱 → 出外事业难遂，注意意外
- 迁移宫化忌 → 出外不顺，加煞防意外灾伤

### 2. 命宫（出行根基）
- 命宫主内部本地，迁移宫主外部外地
- 命强宜本地发展，迁强宜外地发展
- 命宫弱+迁移强 → 出外反而更好
- 命宫强+迁移弱 → 留守本地更有利

### 3. 福德宫（出行动机/心态）
- 出行的心理驱动力
- 外出期间的精神状态
- 福德宫好 → 外出享受愉快
- 福德宫差 → 外出精神压力大

### 4. 夫妻宫（出行伴侣）
- 出行是否有伴侣
- 异地恋情/异地婚姻
- 夫妻宫+迁移宫合参

### 5. 田宅宫（出行离开的家）
- 离开后的家宅情况
- 出行归来后的状态
- 田宅宫好 → 出行无忧，家有保障

## 二、必须检查的星曜

### 出行吉星
- **天马**：出行之星，主动中取财、动中发展
- **禄存+天马（禄马交驰）**：最佳出行组合，越跑越赚
- **化禄入迁移**：外出顺利，有财禄
- **天魁/天钺入迁移**：外出遇贵人
- **左辅/右弼入迁移**：外出有人帮助

### 出行凶星
- **迁移宫化忌**：出外不顺，加煞防意外
- **擎羊+蜚廉+白虎+破碎+阴煞+地劫**聚会迁移宫 → 车祸信号
- **化忌+天机+陀罗+天马+巨门**聚会迁移宫 → 交通意外
- **迁移宫空劫**：出行虚耗，劳而无功

### 主星在迁移宫的含义
- **紫微在迁移**：出外有贵人，得人尊重
- **天机在迁移**：出外多变，灵活机动
- **太阳在迁移**：出外光明，适合公众领域
- **武曲在迁移**：出外求财，适合金融竞争
- **天同在迁移**：出外享受，适合旅游休闲
- **廉贞在迁移**：出外重规矩，注意法律问题
- **天府在迁移**：出外稳定，有积蓄
- **太阴在迁移**：出外低调，暗中得利
- **贪狼在迁移**：出外多桃花多交际
- **巨门在迁移**：出外多口舌是非
- **天相在迁移**：出外有人帮助
- **天梁在迁移**：出外逢贵人长辈
- **七杀在迁移**：出外独立奋斗，有波折
- **破军在迁移**：出外变化大，不安定

## 三、四化判断规则（严格遵守）

### 化禄入迁移宫
- 出行顺利，外地发展好运
- 适合在该年外出发展或旅行
- 外出有财禄

### 化权入迁移宫
- 在外地掌握主动权
- 适合在外地争取领导地位
- 外出有竞争力

### 化科入迁移宫
- 因外出得名声
- 外出遇贵人
- 适合学术交流、参展

### 化忌入迁移宫
- 出外不顺，注意安全
- 该年不宜远行
- 加煞时防意外灾伤

### 特殊四化
- **命宫化忌冲迁移宫** → 自身限制出行发展
- **迁移宫化忌冲命宫** → 外出遭遇影响自身
- **福德宫化忌冲迁移宫** → 心态不好影响出行体验
- **夫妻宫化禄入迁移宫** → 因感情外出或异地恋

## 四、行运判断规则

### 大限出行运
- 大限走迁移宫 → 该十年出行运被引动
- 大限化禄入迁移宫 → 十年外出发展好运
- 大限化忌入迁移宫 → 十年外出不顺

### 流年出行运
- 流年化禄入迁移宫 → 今年出行顺利
- 流年化忌入迁移宫 → 今年出行不顺
- 流年天马入迁移宫 → 今年有出行机会

### 出行安全判断
- 迁移宫化忌+煞 → 出行有安全风险
- 流年化忌入迁移+凶星聚会 → 该年出行需特别注意安全
- 大限+流年双重化忌入迁移 → 出行风险极高

### 宜出外信号
- 迁移宫强于命宫
- 命宫有天马星
- 机阳巨同宫（异乡发展格）
- 流年化禄入迁移

### 不宜出外信号
- 迁移宫化忌加煞
- 迁移宫有擎羊白虎破碎聚会
- 命宫强于迁移宫
- 流年化忌入迁移

## 五、特殊格局判断

### 出行吉格
- **禄马交驰**（禄存与天马同对宫） → 出行生财
- **迁移宫庙旺+吉星** → 出外有贵人
- **命迁线双旺** → 内外皆通
- **天马+化禄** → 动中得财

### 出行凶格
- **迁移宫化忌+煞聚会** → 出行灾伤
- **迁移宫空劫** → 出行虚耗
- **迁移宫擎羊+白虎+破碎** → 车祸风险

## 六、注意事项

### 易错点
- **命迁对比是关键**：不是迁移好就一定要出外，要看和命宫的对比
- **天马不一定是出行**：天马也可能表示内心想动而非实际出行
- **迁移宫凶≠不能出门**：只表示运势不顺，不是绝对禁止

### 禁止事项
- **禁止断车祸**：只提示出行安全风险，不能预测具体事故
- **禁止鼓励冒险出行**：迁移宫化忌+煞时必须提示安全风险
- **禁止给具体出行日期**：只指出有利/不利的运势窗口
