# 学业解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 官禄宫/事业宫（最核心）
- 学业发展、学习能力、考试运势
- 官禄宫在学业分析中代表学业宫
- 官禄宫旺+吉星 → 学业顺利
- 官禄宫化忌 → 学业受阻

### 2. 迁移宫（异地求学）
- 涉及异地/留学时兼看迁移宫
- 迁移宫强 → 适合在外地求学
- 迁移宫弱 → 本地学业更适宜

### 3. 田宅宫（学习环境）
- 备考需要集中精力时兼看田宅宫
- 田宅宫好 → 学习环境好
- 田宅宫不好 → 学习环境差，影响备考

### 4. 父母宫（才智遗传/师长缘）
- 头脑才智和遗传因素
- 与老师/长辈的缘分
- 父母宫有机阴府相昌曲魁钺化科等智能星 → 智力好

### 5. 命宫（学习根基）
- 自身学习能力的基础
- 命宫旺 → 学习有天赋
- 命宫弱 → 需加倍努力

## 二、必须检查的星曜

### 智能星（学业吉利星曜）
- **文昌**：文学才华、考试运好
- **文曲**：才艺出众、口才好
- **天魁/天钺**：贵人助考、机会好
- **化科**：名声、文凭、学术成就

### 各主星在学业中的含义
- **紫微**：学习掌控力强，有望成为佼佼者
- **天机**：思维敏捷，举一反三，但易浮躁
- **太阳**：学习热情高，容易获认可
- **武曲**：学习执行力强，按部就班
- **天同**：学习轻松享受，但缺乏竞争意识
- **廉贞**：学习守规矩，品学兼优
- **天府**：学习稳健，成绩稳定
- **太阴**：学习细腻，适合文科
- **贪狼**：多才多艺，兴趣广泛但不够专一
- **巨门**：善于思辨分析，适合研究
- **天相**：学习有辅助，需引导
- **天梁**：正直好学，适合教育领域
- **七杀**：学习冲劲足，但易走极端
- **破军**：学习创新，但容易破坏秩序

### 不利学业的星曜
- **化忌入官禄** → 学业受阻，考试不利
- **昌曲化忌** → 文才受损，考试失利
- **空劫入官禄** → 学业虚浮，难以专注
- **煞星聚于官禄** → 学业波折多

## 三、四化判断规则（严格遵守）

### 化禄入官禄宫
- 学业顺利，学习状态好
- 适合在该年参加重要考试

### 化权入官禄宫
- 学习掌控力强，适合竞争性考试
- 可能过于执拗

### 化科入官禄宫
- 学业有成就，获认可
- 考试运好，名次提升
- **化科是学业最吉的四化**

### 化忌入官禄宫
- 学业受阻，考试不顺
- 注意学习方法和心态

### 昌曲四化对学业的影响
- **文昌化科/文曲化科** → 考试极佳
- **文昌化忌/文曲化忌** → 文才受损，考试失利
- **化科入官禄+昌曲吉** → 最佳考试组合

### 特殊四化
- **父母宫化科** → 师长认可，学业有助力
- **命宫化忌冲官禄** → 自身情绪影响学业
- **迁移宫化禄入官禄** → 外出求学有利

## 四、行运判断规则

### 大限学业
- 大限走官禄宫 → 该十年学业被引动
- 大限化科入官禄宫 → 十年学业好
- 大限化忌入官禄宫 → 十年学业波折

### 流年学业
- 流年化科入官禄宫 → 最佳考试年
- 流年化禄入官禄宫 → 学习顺利年
- 流年化忌入官禄宫 → 考试不利年
- 流年昌曲化科 → 文考大利

### 考试时机判断
- 官禄宫旺 + 流年化科入官禄 → 考试最佳窗口
- 昌曲魁钺入官禄 + 无煞冲破 → 必名列前茅
- 阳梁昌禄格成立 → 科名第一

## 五、特殊格局判断

### 学业吉格
- **阳梁昌禄格** → 科名第一，考试极佳
- **昌曲魁钺入官禄** → 考试运好
- **官禄宫庙旺+化科** → 学业有成
- **文星拱命格** → 文才出众

### 学业凶格
- **官禄宫化忌+煞** → 学业严重受阻
- **昌曲化忌** → 文才受损
- **智能星化忌+羊陀夹忌** → 智力受影响

### 弱智信号
- 命宫和父母宫无智能星（机阴府相昌曲魁钺化科）
- 智能星化忌加煞
- 羊陀夹忌时更重

## 六、注意事项

### 易错点
- **学业≠事业**：学业分析以官禄宫为学业宫，但侧重学习能力和考试运
- **异地因素**：留学/异地求学必须兼看迁移宫
- **备考环境**：大型考试需看田宅宫（学习环境）

### 禁止事项
- **禁止给具体分数预测**：只说考试运势方向
- **禁止断定能否考上**：只说有利/不利的时机
- **禁止忽视努力因素**：命理分析是辅助，不替代学习
