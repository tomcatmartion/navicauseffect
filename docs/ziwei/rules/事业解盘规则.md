# 事业解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 官禄宫（最核心）
- 事业性质、职业走向、升迁机会
- 事业发展层次和工作稳定性
- 官禄宫强 → 事业发达，宜公职或合伙
- 官禄宫弱 → 事业不稳，多有变动

### 2. 命宫（事业上限）
- 自身能力与格局，决定事业天花板
- 命宫好 + 官禄宫好 → 可从政，权势之官
- 命宫好 + 财帛宫强 → 可做老板
- 命宫无主星 → 需借对宫判断，事业相对弱化

### 3. 迁移宫（外在机遇）
- 社会活动能力、贵人提携
- 迁移宫强 → 宜外地发展，适合社交性职业
- 迁移宫弱 → 出外事业难遂

### 4. 财帛宫（事业财）
- 事业带来的收入实际情况
- 财帛宫强于官禄宫 → 宜从商
- 官禄宫强于财帛宫 → 宜公职

### 5. 福德宫（事业理念）
- 事业开创性、做事理念
- 与官禄宫合参判断事业格局

## 二、必须检查的星曜

### 官禄宫坐星 → 职业方向

#### 宜公务文职
- **紫微**：领导管理、政界
- **太阳**：能源、电力、律师、司法、政界
- **天梁**：教育、纪检监察、理论、宗教
- **天相**：行政、秘书、公职
- **文昌/文曲**：文职、文书、媒体
- **左辅/右弼**：辅助管理、行政
- **天魁/天钺**：公职、贵人带官

#### 宜商界
- **武曲**：金融财经、钢铁矿业、军警
- **廉贞**：政界（吉时）、电脑电器、法律
- **天府**：财务管理、银行
- **太阴**：财务、房产、餐饮、护理
- **贪狼**：交际类、娱乐业、餐饮
- **七杀**：武职、竞争性行业
- **破军**：开创性行业、变革型职业

#### 宜技术
- **天机**：机械、计算机、策划统计
- **文昌/文曲**：写作、研究、技术

#### 宜五术宗教
- **天巫/华盖**：宗教、命理
- **天机/天梁/贪狼**：五术相关

### 事业宫辅星影响
- **化科**：学识才华得认可，适合专业路线
- **化权**：职场掌权，适合管理或创业
- **化禄**：事业顺遂，收入稳增
- **化忌**：事业受阻，不宜从政

## 三、四化判断规则（严格遵守）

### 化禄入官禄宫
- 事业顺遂，机会多，收入稳定增长
- 适合在本行深耕

### 化权入官禄宫
- 职场中掌握主动权
- 适合创业或争取晋升
- 可能过于强势，注意人际关系

### 化科入官禄宫
- 学识才华得到认可
- 适合专业技术路线
- 因名声得事业机会

### 化忌入官禄宫
- 事业受阻，注意职场人际和工作变动
- 不宜从政和给人打工
- 化忌之年容易换工作或事业受挫

### 官禄宫自化忌
- 事业难以稳定，频繁换工作
- 自己管不好职业发展

### 命宫化忌冲官禄宫
- 自身能力或性格影响事业发展
- 需先解决个人问题再谈事业

### 特殊四化
- **官禄宫化禄冲财帛宫** → 事业带动财运
- **财帛宫化忌冲官禄宫** → 收入问题影响事业
- **迁移宫化禄入官禄宫** → 外在机遇助推事业

## 四、行运判断规则

### 大限事业
- 大限宫干化禄入官禄宫 → 十年事业顺遂
- 大限宫干化忌入官禄宫 → 十年事业受阻
- 大限命宫落原局官禄宫 → 该十年事业被引动

### 流年事业
- 流年化禄入官禄宫 → 今年事业顺利
- 流年化忌入官禄宫 → 今年工作变动
- 大限+流年双重化忌入官禄 → 该年事业变动幅度大

### 三层合参
1. 原局官禄宫旺衰 → 事业承载力
2. 大限引动方向 → 十年事业走向
3. 流年四化 → 具体年份的转折点

### 换职业信号
- 官禄宫主星为杀破狼 → 工作行业变化大
- 大限化忌入官禄宫 → 宜上班不宜从商
- 流年化忌入官禄宫 → 可能换工作

## 五、特殊格局判断

### 事业吉格
- **紫府朝垣** → 权贵格局，事业通达
- **府相朝元** → 强大事业格局
- **阳梁昌禄** → 科甲名声，适合公职文教
- **机月同梁** → 适合公务员、企划类
- **日照雷门**（太阳在卯宫入命） → 事业光明

### 事业凶格
- **官禄宫逢四煞** → 事业多波折
- **空劫入官禄** → 事业浮动，不适合创业
- **化忌加煞入官禄** → 事业严重受阻
- **命宫化忌冲官禄** → 事业受自身拖累

### 格局成立条件
- 宫位V3.0定性 ≥ 🟡磨炼 → 格局可能成立
- 宫位V3.0定性 < 🟡磨炼 → 格局虚华，护佑破损

## 六、注意事项

### 易错点
- **煞星双面性**：官禄宫有煞 → 宜武职（竞争性职业），不一定差
- **空宫处理**：官禄宫空宫 → 借对宫，事业方向不确定
- **行业判断**：以财帛宫干飞四化，化忌落宫主星代表的行业准确率较高

### 禁止事项
- **禁止只看官禄宫**：必须结合命宫格局判断事业上限
- **禁止在煞忌聚集时给乐观判断**：化忌+煞星聚集的事业宫不能说升迁
- **禁止混淆"化忌入官禄宫"和"官禄宫化忌"**：前者是外力破坏事业，后者是自身事业不稳
- **禁止忽视迁移宫**：很多事业的转折点在迁移宫
- **禁止给具体职位预测**：只能判断方向和类型
