## 🎯 V3.0评分等级与星曜赋性的对应

### 等级定义与对应

| V3.0定性 | 分数段 | 星曜赋性表现 | 事项分析含义 | 查库关键字 |
|---------|--------|------------|-----------|---------|
| 🟢 实旺 | 7.5～8.5 | **旺宫正面发挥** | 承载力强，制煞为用充分 | `旺宫表现` |
| 🟢 实旺偏磨炼 | 6.0～7.4 | **旺宫正面发挥**（略有摩擦） | 整体强旺，有小幅阻力 | `旺宫表现` |
| 🟡 磨炼 | 4.5～5.9 | **平宫中性呈现** | 能量中性，需积累 | `平宫表现` |
| 🟠 虚浮 | 3.0～4.4 | **陷宫负面放大** | 引爆阈值低，虚华 | `陷宫表现` |
| 🔴 凶危 | 1.5～2.9 | **陷宫逞凶** | 吉星为诱饵，经不起忌穿透 | `陷宫表现` |
| ⚫ 绝败 | < 1.5或强制触发 | **无法分析** | 结构性毁灭，无救 | 跳过分析 |

---

### 详细对应说明

#### 🟢 实旺（7.5～8.5）

**V3.0定性特点**：
- Final_Score ≥ 7.5
- 承载力极强
- 制煞为用可充分发挥
- 即使有煞星也呈现正面特质

**查星曜赋性库时**：
- 查找 **`旺宫表现`** 列
- 星曜展现正面特质
- 吉星真吉，煞星也有正面含义

**事项分析时**：
- 原局底盘定性：**强旺顺遂**
- 格局护佑：完整有效
- 可充分发挥主星特性
- 制煞机制可正常运作

**示例**：
```
紫微在求学·事业宫，Final_Score = 7.8（🟢实旺）

查库：紫微→求学→事业宫→【旺宫表现】
结论：领导力强，学习掌控力强，自律性高，有望成为佼佼者

分析：原局此宫强旺，格局保护完整
```

---

#### 🟢 实旺偏磨炼（6.0～7.4）

**V3.0定性特点**：
- Final_Score 6.0～7.4
- 整体强旺
- 有小幅摩擦或临界状态
- 大限吉化可激活制煞为用

**查星曜赋性库时**：
- 查找 **`旺宫表现`** 列
- 星曜仍为正面，但有小幅限制
- 需注意临界状态标注

**事项分析时**：
- 原局底盘定性：**强旺，有磨炼**
- 格局护佑：基本完整，但有临界
- 大限吉化时可激活，忌化时需留心
- 本身仍有较强承载力

---

#### 🟡 磨炼（4.5～5.9）

**V3.0定性特点**：
- Final_Score 4.5～5.9
- 能量中性
- 后天爆发空间大
- 行运影响显著

**查星曜赋性库时**：
- 查找 **`平宫表现`** 列
- 星曜呈现中性，吉不完整，煞不极端
- 需要外力推动才能充分展现

**事项分析时**：
- 原局底盘定性：**中等有阻**
- 格局护佑：一般
- 需持续积累和推动
- 大限吉化效果明显

**示例**：
```
武曲在求财·财帛宫，Final_Score = 5.2（🟡磨炼）

查库：武曲→求财→财帛宫→【平宫表现】
结论：执行力中等，需要坚持和积累

分析：原局此宫平稳，需大限吉化激活
```

---

#### 🟠 虚浮（3.0～4.4）

**V3.0定性特点**：
- Final_Score 3.0～4.4
- 引爆阈值低
- 吉星呈虚华
- 经不起忌穿透

**查星曜赋性库时**：
- 查找 **`陷宫表现`** 列
- 星曜负面放大
- 吉星虚华不实
- 需标注虚华警告

**事项分析时**：
- 原局底盘定性：**虚浮困难**
- 格局护佑：不完整，需警惕
- 吉星常为诱饵
- 凶限容易穿透

**标注方式**：
```
虚浮宫位：星曜X（虚华）
含义：该星曜呈现虚浮特性，需警惕
```

---

#### 🔴 凶危（1.5～2.9）

**V3.0定性特点**：
- Final_Score 1.5～2.9
- 防御虚弱
- 吉星为诱饵
- 经不起忌煞交冲

**查星曜赋性库时**：
- 查找 **`陷宫表现`** 列
- 星曜逞凶
- 所有特性都负面放大

**事项分析时**：
- 原局底盘定性：**凶危高风险**
- 格局护佑：破损，无底线
- 需要提前预警
- 大限吉化常被穿透

---

#### ⚫ 绝败（<1.5 或强制触发）

**V3.0定性特点**：
- Final_Score < 1.5 或强制触发绝败
- 结构性毁灭
- 无法分析

**星曜赋性库**：
- **跳过查询**，无需分析星曜含义

**事项分析时**：
- **跳过分析**
- 告知命主此宫无救
- 重点在其他宫位

## 📋 事项分析逻辑中的V3.0定性应用

### 原局底盘分析阶段

**第一步：确认宫位旺弱定性**
```
使用V3.0的Final_Score定性（而非模糊描述）

输入：宫位名 + Final_Score（数值）+ 定性（颜色）
例：财帛宫 3.2 🟠虚浮
```

**第二步：识别格局护佑机制**
```
根据宫位定性判断护佑机制能否成立：

🟢实旺 → 护佑格局完整有效
🟡磨炼 → 护佑格局基本有效
🟠虚浮 → 护佑格局易破损
🔴凶危 → 护佑格局无效
⚫绝败 → 无需判断
```

**第三步：事项宫位四维合参**
```
依据V3.0的定性（而非单纯的星曜旺衰）进行四维合参

例：
• 本宫：Final_Score = 5.2（🟡磨炼）→ 中性承载
• 对宫：Final_Score = 7.1（🟢实旺）→ 加强本宫
• 合宫：Final_Score = 3.8（🟠虚浮）→ 支撑不足
• 临宫：Final_Score = 6.5（🟢实旺）→ 适度制约

综合定性：中等有阻（原局不够稳，但对宫临宫支持）
```

### 行运分析阶段

**大限引动判断**
```
大限宫干四化引动 ×（原宫位的V3.0定性）= 效果

吉化引动旺宫(🟢) → 吉象充分激活
吉化引动平宫(🟡) → 吉象需大限推动
吉化引动虚浮(🟠) → 吉象虚华但可救济
忌化引动凶危(🔴) → 凶象直接穿透
```

### 流年引动阶段

**流年×大限×原局定性的三层判断**
```
流年吉化 × 大限方向 × 原宫位定性 = 今年引动

例：
流年吉 + 大限吉 + 原宫🟢旺 = 最佳推进窗口
流年吉 + 大限吉 + 原宫🟠虚 = 虚化救济（效果打折）
流年忌 + 大限凶 + 原宫🔴凶 = 风险最高
```

## 🔗 查询流程示例

### 场景：命主求财

**步骤1：打分**
```
用V3.0给财帛宫打分
结果：财帛宫 Final_Score = 5.3，定性 🟡磨炼
```

**步骤2：查星曜赋性**
```
财帛宫主星：武曲
定性：🟡磨炼 → 查【平宫表现】

查库路径：
KB_星曜赋性_V2.0
→ 【求财】章节
→ 【财帛宫】
→ 【武曲】
→ 【平宫表现】：执行力中等，需坚持...
```

**步骤3：事项分析**
```
原局底盘定性：
• 财帛宫定性为🟡磨炼（中等有阻）
• 需查对宫/三合宫定性（四维合参）
• 判断格局护佑（能否成立）

最终：原局中等有阻，需大限吉化推动
```

**步骤4：行运推演**
```
查看命主当下大限：
大限宫干四化 → 是否引动财帛宫
财帛宫定性🟡 + 大限吉化 → 激活路径
```

## ⚠️ 使用禁令

| 禁令 | 说明 |
|------|------|
| 禁止跳过V3.0定性 | 必须先用V3.0打分，才能查赋性库和做事项分析 |
| 禁止混用定性标准 | 不能同时用V3.0和其他标准混淆 |
| 禁止忽视分数段 | 7.5和6.1都是旺，但含义不同（一个实旺，一个磨炼） |
| 禁止绝败宫位做分析 | 绝败宫位无需继续分析，直接告知无救 |
| 禁止虚浮当旺论 | 虚浮宫位的吉星是虚华，不能当真吉论 |
| 禁止忽视临界状态 | 临界宫位（如6.2分）需特别标注 |
