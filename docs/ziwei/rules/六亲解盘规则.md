# 六亲解盘规则

## 一、必须检查的宫位（按优先级）

六亲分析需根据具体问的亲缘关系选择主看宫位：

### 1. 父母（主看父母宫）
- 父母的性情才能、与父母的缘份
- 遗传优劣、文化程度
- 父母宫好 → 有靠山，少年得照顾
- 父母宫不好 → 前半生劳心劳力
- 兼看：太阳（父星）、太阴（母星）的旺衰

### 2. 兄弟姐妹（主看兄弟宫）
- 兄弟姐妹的容貌性情、关系好坏
- 兄弟宫吉星多煞少 → 关系好有助益
- 兄弟宫煞忌多吉少 → 关系不好无助益
- 兼看：天机星

### 3. 朋友/下属（主看奴仆宫）
- 朋友、同事、下属的关系
- 人际关系和人缘
- 奴仆宫不好 → 易被小人欺骗
- 走到奴仆宫的岁限 → 人灾预警

### 4. 配偶（主看夫妻宫）
- 详见感情解盘规则
- 男兼看太阴（妻星），女兼看太阳（夫星）

### 5. 子女（主看子女宫）
- 详见子女解盘规则

### 6. 上司/靠山（看父母宫+迁移宫）
- 父母宫 → 有无靠山
- 迁移宫 → 社会贵人
- 官禄宫 → 与上司关系

## 二、必须检查的星曜

### 六亲宫位主星影响

#### 吉星入六亲宫
- **紫微**：该六亲有地位有能力
- **天府**：该六亲稳重，有财
- **天相**：该六亲温和，有帮助
- **天同**：该六亲温柔，关系好
- **天梁**：该六亲正直，如长辈般关怀
- **化禄**：该六亲带来好运和助力
- **天魁/天钺**：该六亲是贵人

#### 凶星入六亲宫
- **七杀/破军**：该六亲个性强，关系紧张
- **廉贞**：该六亲原则性强，易有纠纷
- **巨门**：与该六亲口角是非多
- **化忌**：与该六亲关系有阻碍
- **擎羊**：与该六亲冲突激烈
- **陀罗**：与该六亲纠缠不清
- **空劫**：与该六亲缘薄，聚少离多

### 太阳太阴对六亲的影响
- **太阳庙旺**：男命父好，女命夫好
- **太阳失陷**：男命与父缘薄，女命与夫缘薄
- **太阴庙旺**：男命妻好，女命母好
- **太阴失陷**：男命与妻缘薄，女命与母缘薄

## 三、四化判断规则（严格遵守）

### 化禄入六亲宫
- 与该六亲关系好，能得助力
- 该六亲给你带来好运

### 化权入六亲宫
- 该六亲强势，关系中有权力角力
- 需适当退让

### 化科入六亲宫
- 该六亲有才华有名声
- 因该六亲而得名声

### 化忌入六亲宫
- 与该六亲关系有阻碍
- 该六亲给你带来麻烦
- 需注意该年与该六亲的矛盾

### 特殊四化
- **命宫化忌冲兄弟宫** → 自身影响兄弟姐妹关系
- **兄弟宫化忌冲命宫** → 兄弟姐妹给你带来困扰
- **父母宫化忌冲疾厄宫** → 遗传问题
- **奴仆宫化忌冲命宫** → 小人损害

## 四、行运判断规则

### 大限六亲运
- 大限走六亲宫 → 该十年与该六亲关系被引动
- 大限化禄入六亲宫 → 十年关系好
- 大限化忌入六亲宫 → 十年关系紧张

### 流年六亲运
- 流年化禄入六亲宫 → 今年关系融洽
- 流年化忌入六亲宫 → 今年关系有矛盾
- 流年凶星入奴仆宫 → 今年注意小人

### 人灾预警
- 奴仆宫不好 → 岁限走到时易被小人欺负骗钱
- 流年化忌入奴仆宫+煞 → 今年人际风险

## 五、特殊格局判断

### 六亲吉格
- 六亲宫庙旺+吉星 → 与该六亲关系好
- 六亲宫化禄+贵人星 → 该六亲是贵人

### 六亲凶格
- 六亲宫逢四煞 → 关系紧张
- 六亲宫化忌+煞 → 刑克该六亲
- 六亲宫空劫 → 缘薄

## 六、注意事项

### 易错点
- **六亲宫空宫**：借对宫，关系不确定
- **煞星双面性**：兄弟宫有煞不一定关系差，可能竞争中有助益
- **太阳太阴性别差异**：男女命对太阳太阴的解读不同

### 禁止事项
- **禁止断六亲生死**：只说关系好坏和缘分深浅
- **禁止挑拨六亲关系**：只说相处建议，不评判对错
- **禁止负面评价他人**：说"关系需经营"而非"对方不好"
