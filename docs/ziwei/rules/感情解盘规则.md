# 感情解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 夫妻宫（最核心）
- 配偶质量、婚姻吉凶、感情走向
- 夫妻宫星曜庙旺吉多 → 配偶好，婚姻和谐
- 夫妻宫失陷煞多 → 配偶差，婚姻不美
- 夫妻宫强于命宫 → 可得益婚姻（对方条件优于自己）
- 夫妻宫弱 → 婚姻不美，需经营

### 2. 命宫（自身条件）
- 命宫有不利婚姻的星（杀破廉贪武巨煞忌）→ 自身影响婚姻
- 命宫格局决定你在感情中的角色
- 命宫与夫妻宫强弱对比 → 谁主导关系

### 3. 福德宫（感情感受/现金流）
- 感情生活品质、精神满足度
- 感情走向讨论时兼看福德宫
- 福德宫也与生活质量、现金流相关

### 4. 迁移宫（社交缘/外在缘分）
- 社交活动中的桃花机遇
- 外出认识的缘分
- 配偶是否为外地人

### 5. 田宅宫（家庭基础）
- 婚后家庭生活
- 感情是否影响家庭稳定
- 与子女宫合参看家庭完整度

### 6. 太阳/太阴（男看妻星/女看夫星）
- 男命兼看太阴旺衰 → 太阴失陷不利妻
- 女命兼看太阳旺衰 → 太阳失陷不利夫
- 太阳/太阴庙旺 → 对另一半有利

## 二、必须检查的星曜

### 夫妻宫主星与感情特质

#### 吉星入夫妻宫
- **紫微**：配偶地位高，有主见，但有控制欲
- **天府**：配偶稳重，善理财，家庭观念强
- **天相**：配偶温和体贴，但缺乏主见
- **天同**：配偶温柔浪漫，感情好，但可能缺乏上进心
- **天梁**：配偶正直善良，年龄差距可能较大
- **化禄/化科**：感情顺利，配偶条件好

#### 复杂星入夫妻宫
- **天机**：配偶聪明但多变化，感情有波折
- **太阳**（旺）：配偶阳光开朗；失陷则不利
- **太阴**（旺）：配偶温柔细腻；失陷则不利
- **武曲**：配偶刚强独立，感情需磨合
- **贪狼**：配偶多才多艺但桃花重，需看辅星

#### 凶星入夫妻宫
- **七杀**：配偶个性强，冲突多，宜晚婚
- **破军**：感情波折大，多变动，配偶不安分
- **廉贞**：感情纠葛，加煞更凶
- **巨门**：口角是非多，猜忌，需沟通

### 桃花星
- **贪狼**：正桃花星，才艺与魅力
- **廉贞**：次桃花星，感情纠葛
- **文昌/文曲**：异性缘好
- **天姚/红鸾/天喜**：桃花年运星

### 煞忌对感情的影响
- **擎羊**：感情冲突激烈
- **陀罗**：感情纠缠不清
- **火星/铃星**：感情急躁/暗恨
- **化忌**：感情受挫、分离
- **空劫**：感情虚化，聚少离多

## 三、四化判断规则（严格遵守）

### 化禄入夫妻宫
- 感情顺利，婚姻美满
- 适合在该年确定关系或结婚
- 配偶带来好运

### 化权入夫妻宫
- 感情中掌握主导权
- 配偶强势，需磨合
- 适合在关系中争取主动

### 化科入夫妻宫
- 因名声或学识得良缘
- 感情优雅，受人祝福
- 配偶有才华

### 化忌入夫妻宫
- 感情受挫，容易分离
- 该年不宜做重大感情决定
- 注意第三者介入

### 夫妻宫自化忌
- 感情反复，自己在关系中制造问题
- 需反思自身行为

### 特殊四化
- **命宫化忌冲夫妻宫** → 自身性格伤害感情
- **夫妻宫化忌冲命宫** → 配偶给你带来伤害
- **迁移宫化禄入夫妻宫** → 外出社交带来良缘
- **福德宫化忌冲夫妻宫** → 精神状态影响感情

## 四、行运判断规则

### 大限感情
- 大限走夫妻宫 → 该十年感情被引动
- 大限化禄入夫妻宫 → 十年感情运好
- 大限化忌入夫妻宫 → 十年感情波折

### 流年感情
- 流年化禄入夫妻宫 → 今年桃花旺，适合恋爱结婚
- 流年化忌入夫妻宫 → 今年感情波折
- 流年红鸾/天喜入命或夫妻宫 → 桃花年

### 迟早婚判断
- 夫妻宫无主星或逢空星 → 晚婚
- 命宫有杀破廉贪武巨煞忌 → 晚婚免刑克
- 刚烈孤克之星 → 早婚有克，宜晚婚
- 夫妻宫吉星多 → 可早婚

## 五、特殊格局判断

### 感情吉格
- **夫妻宫庙旺+吉星** → 婚姻和谐
- **桃花星+正气吉星** → 自由恋爱，感情好
- **太阳太阴庙旺** → 夫妻恩爱

### 感情凶格
- **夫妻宫逢四煞之一** → 刑克，见六煞定有刑克
- **夫妻宫化忌加煞** → 感情破裂
- **夫妻宫加陀罗/见羊陀** → 感情纠缠，离婚风险
- **命宫失陷+夫妻宫失陷+加煞忌** → 严重刑克

### 六吉单星特殊规则
- 六吉星（左辅右弼文昌文曲天魁天钺）单星独守夫妻宫
- 易有二婚或感情困扰
- 年龄相差悬殊
- 易有第三者
- 不喜六吉单星同宫夫妻宫

### 刑克信号（严重）
- 夫妻宫逢恶星与煞刑忌凶曜聚会 → 丧配
- 男命夫妻宫不好+太阴失陷+三方见煞 → 丧妻
- 女命夫妻宫不好+太阳失陷+三方见煞 → 丧夫
- 夫妻宫见化忌加煞 → 刑克

## 六、注意事项

### 易错点
- **桃花双面性**：桃花星+正气吉星是好事，桃花星+煞星才是感情纠纷
- **六吉单星**：看似好星，独守夫妻宫反而不美
- **空宫处理**：夫妻宫空宫 → 晚婚，借对宫但效果打折

### 禁止事项
- **禁止只看夫妻宫**：必须兼看命宫和太阳/太阴
- **禁止直接断离婚**：只能说"婚姻波折"或"需经营"，不给极端结论
- **禁止忽视性别差异**：男命看太阴（妻星），女命看太阳（夫星）
- **禁止轻率断第三者**：需多个信号同时出现才可提及
- **禁止给具体结婚年份**：只能指出桃花旺的时间窗口
