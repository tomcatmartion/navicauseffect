# 子女解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 子女宫（最核心）
- 子女的人数、容貌、性情、才能
- 与子女的感情关系
- 子女宫吉星多 → 子女多且好
- 子女宫煞多 → 子女少或不肖

### 2. 田宅宫（子孙命运）
- 田宅宫好坏直接影响子孙
- 子女宫不好的原因与田宅有关
- 田宅不好 → 难得好子孙

### 3. 夫妻宫（生育基础）
- 婚姻关系影响子女成长环境
- 夫妻宫差+子女宫差 → 家庭环境不利子女

### 4. 命宫（自身条件）
- 命主的精力和能力影响养育质量
- 命宫弱 → 抚养子女力不从心

## 二、必须检查的星曜

### 子女宫主星与子女特质

#### 吉星入子女宫
- **紫微**：子女贵气，有领导力
- **天府**：子女稳重，善理财
- **天相**：子女温和，有辅佐之才
- **天同**：子女温柔，但可能缺乏进取心
- **天梁**：子女正直善良，年龄差距大
- **化禄/化科**：子女聪明，有出息

#### 变动星入子女宫
- **天机**：子女个性多变，有分离倾向
- **太阳**（旺）：子女阳光开朗
- **太阴**（旺）：子女温柔细腻

#### 凶星入子女宫
- **七杀**：子女性格刚烈，叛逆
- **破军**：子女不安分，变动多
- **廉贞**：子女任性，加煞更凶
- **巨门**：与子女沟通不畅，有代沟

### 影响子女人数的星曜
- **空劫/大耗** → 子女少或与子女缘薄
- **化忌** → 与子女关系有阻碍
- **吉星多** → 子女多且好
- **煞多** → 子女少或不肖

### 特殊星曜组合
- **子女宫有天机** → 子女个性多变，有分离
- **子女宫化忌+煞** → 生育困难或子女缘薄
- **桃花星入子女宫** → 子女有艺术天赋

## 三、四化判断规则（严格遵守）

### 化禄入子女宫
- 子女运好，子女聪明有出息
- 与子女关系融洽
- 适合在该阶段考虑生育

### 化权入子女宫
- 子女有主见、个性强
- 需注意不要过度控制
- 子女有掌权潜力

### 化科入子女宫
- 子女有才华、学业好
- 因子女而得名声
- 子女在学业上有成就

### 化忌入子女宫
- 与子女关系有阻碍
- 生育可能不顺
- 子女教育需多费心

### 特殊四化
- **田宅宫化忌冲子女宫** → 家庭环境不利子女成长
- **夫妻宫化忌冲子女宫** → 夫妻关系影响子女
- **命宫化忌冲子女宫** → 自身原因导致与子女缘薄

## 四、行运判断规则

### 大限子女运
- 大限走子女宫 → 该十年子女运被引动
- 大限化禄入子女宫 → 十年子女运好
- 大限化忌入子女宫 → 十年子女关系需注意

### 流年子女运
- 流年化禄入子女宫 → 今年子女缘好
- 流年化忌入子女宫 → 今年与子女有矛盾
- 流年红鸾入子女宫 → 可能有生育机会

### 生育时机判断
- 子女宫旺 + 流年化禄入子女 → 良好生育时机
- 子女宫弱 + 流年化忌入子女 → 生育需谨慎
- 田宅宫吉 + 子女宫吉 → 家庭稳定适合生育

## 五、特殊格局判断

### 子女吉格
- 子女宫庙旺+吉星 → 子女优秀
- 子女宫化禄/化科 → 子女有出息
- 田宅宫+子女宫都吉 → 子孙兴旺

### 子女凶格
- 子女宫逢空劫 → 子女缘薄
- 子女宫化忌+煞 → 子女不肖或关系差
- 田宅宫凶+子女宫凶 → 子孙不旺

## 六、注意事项

### 易错点
- **子女宫空宫**：借对宫星曜，子女缘不确定
- **天机多变**：不代表不好，只是子女个性灵活
- **煞星不等于无子女**：只表示养育过程辛苦

### 禁止事项
- **禁止分析怀孕求子**：直接拒绝，超出命理分析范围
- **禁止断定子女性别**：只说特质倾向
- **禁止给具体生育时间**：只指出有利的运势窗口
- **禁止负面评价子女**：只说特质，不说"不孝""没出息"
