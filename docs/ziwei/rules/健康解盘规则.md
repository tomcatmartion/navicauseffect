# 健康解盘规则

## 一、必须检查的宫位（按优先级）

### 1. 疾厄宫（最核心）
- 身体健康、先天体质、疾病倾向
- 先审疾厄宫星曜善恶与庙陷
- 疾厄宫凶 → 注意对应五行系统的疾病

### 2. 命宫（健康根基）
- 全面性灾疾的判断基础
- 尤其生死安危必须参看命宫
- **命宫吉强**：即使疾厄宫有凶星，也不一定身体差，有病也能快治好
- **命宫失陷加煞**：必有灾疾

### 3. 父母宫（遗传/先天）
- 遗传疾病、先天体质
- 父母的体质直接影响命主的先天基础
- 父母宫不好 → 先天体质偏弱

### 4. 福德宫（心理/精神）
- 心理疾患、精神状态
- 七情六欲业障病看此宫
- 长期精神压力导致的疾病

### 5. 田宅宫（居住环境）
- 居住环境对健康的影响
- 家宅风水与健康

## 二、必须检查的星曜

### 五行对应疾病

#### 土系（脾胃）
- **紫微**：脾胃、骨骼
- **天府**：脾胃、消化系统
- **天同**：肾脏、水液代谢（兼土水）

#### 木系（肝胆）
- **天机**：神经系统、肝胆
- **贪狼**：肝胆、泌尿系统

#### 金系（肺部/骨骼）
- **武曲**：肺部、骨骼、金属伤害
- **太阴**：内分泌、女性生殖

#### 火系（心血管）
- **太阳**：眼睛、心脏、血液循环
- **廉贞**：血液、心脏、生殖系统
- **火星**：发热、炎症

#### 水系（肾/泌尿）
- **天同**：肾脏
- **巨门**：泌尿系统
- **贪狼**：肾亏

### 凶星对应症

| 凶星 | 对应症状 | 特征 |
|------|---------|------|
| 擎羊 | 血光、外伤、手术 | 尖锐性灾伤，刀伤 |
| 陀罗 | 暗疾、拖延性病症 | 长期慢性病 |
| 火星 | 炎症、意外、高热 | 急性发作 |
| 铃星 | 慢性隐疾、不明原因 | 反复发作 |
| 化忌 | 发病急，疾病加重 | 急性恶化 |
| 空劫 | 虚耗性疾病 | 元气不足 |

### 特殊组合对应症
- **擎羊+太阴同宫** → 肝病
- **贪狼+陀罗同宫** → 肾亏阳萎
- **太阳+火星同宫** → 眼目有伤
- **天同+陀罗** → 目疾或污肿
- **太阴失陷** → 内分泌障碍
- **廉贞+化忌** → 血液疾病

## 三、四化判断规则（严格遵守）

### 化禄入疾厄宫
- 身体健康，体质好
- 即使有病也容易康复

### 化权入疾厄宫
- 体质强壮，但可能过度透支
- 注意工作压力大导致的健康问题

### 化科入疾厄宫
- 找到好的医生或治疗方案
- 适合做养生保健

### 化忌入疾厄宫
- 该年健康运差，易生病
- 对应宫位五行系统的疾病容易发作
- 需提前预防

### 疾厄宫自化忌
- 先天体质偏弱
- 需长期注意养生

### 特殊四化
- **命宫化忌冲疾厄宫** → 自身行为伤害身体（如过劳、饮食不节）
- **福德宫化忌冲疾厄宫** → 精神问题影响身体
- **父母宫化忌冲疾厄宫** → 遗传疾病显现

## 四、行运判断规则

### 大限健康
- 大限走疾厄宫 → 该十年健康被引动
- 大限化忌入疾厄宫 → 十年健康运差，注意预防
- 大限走命宫且化忌入疾厄 → 该十年身体透支

### 流年健康
- 流年化忌入疾厄宫 → 今年健康红灯
- 流年化禄入疾厄宫 → 今年身体好转
- 流年凶星入疾厄宫 → 对应疾病风险

### 疾病轻重判断
- **命宫吉+疾宫凶** → 有病但能治好
- **命宫凶+疾宫凶** → 疾病严重
- **疾宫凶+财官凶** → 事业财运问题的可能性更大
- **疾宫吉+财官凶** → 不是健康问题，是事业财运

## 五、特殊格局判断

### 健康吉格
- 疾厄宫庙旺+吉星 → 先天体质好
- 命宫+疾厄宫都吉 → 身体强健
- 化禄入疾厄+无煞 → 长寿之象

### 健康凶格
- 疾厄宫逢四煞聚会 → 多病多灾
- 疾厄宫化忌+煞 → 重病信号
- 命宫失陷+疾宫失陷+煞 → 健康极差

### 意外伤灾信号
- 命身及父母宫有凶煞 → 伤灾破相
- 擎羊+廉贞 → 血光手术
- 迁移宫擎羊+蜚廉+白虎+破碎+阴煞+地劫+化忌聚会 → 车祸信号

## 六、注意事项

### 易错点
- **疾宫凶≠一定生病**：命宫吉时，疾宫凶可能是事业财运的映射
- **五行对应是基础**：先定五行方向，再细化到具体器官
- **煞星双面性**：庙旺煞星只增加体质的刚强，失陷才是疾病

### 禁止事项
- **禁止给具体疾病诊断**：只能提示倾向，不是医疗建议
- **禁止断寿命**：只说健康趋势，不给寿命预测
- **禁止忽视现代医学**：命理分析是辅助参考，不替代医疗
- **禁止夸大灾厄**：疾宫化忌加煞只能说"需注意"，不能说"必有大病"
