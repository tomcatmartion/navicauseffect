## 第二阶段：原局底盘分析

### 核心原则

> 原局定底盘——先天格局决定这件事的基础能量与护佑机制

### 步骤一：命宫全息底色带入

#### 数据来源
```
来源：SKILL_定性命主性格 的分析结果
```

#### 需要带入的信息

| 项目 | 说明 | 如何使用 |
|------|------|--------|
| **命宫Final_Score** | 命主能级的最终评分 | 决定命主能否承载事项宫位的吉凶 |
| **原局四化**（生年+遁干） | 禄权科忌四化的具体落位 | 禄→顺畅；权→激发；科→名声；忌→阻滞，全程带入分析 |
| **六吉六煞** | 左右禄权科+羊陀火铃 | 具体的助力与风险方向 |
| **命宫性格底色** | 命主的基础性格与行为模式 | 命主应对此事项的基础思维与行为方式 |

#### 操作步骤

1. 打开命主的`SKILL_定性命主性格`分析结果
2. 提取上表中的4个信息项
3. 在这个分析中带入这些信息
4. 用于统领后续所有事项宫位的解读

---

### 步骤二：识别原局关键格局

#### 目标
在分析事项宫位之前，先识别原局是否有影响该事项的关键格局

#### 空宫处理规则

| 情况 | 处理方法 | 说明 |
|------|--------|------|
| **事项宫位空宫** | 借对宫星曜论 | 吉象减半，凶象更凶；意识形态薄弱，需外力推动 |
| **空宫命宫** | 对宫投射为主 | 须看对宫三合构成何种大格局 |
| **空宫本身** | 不等于没结果 | 空宫不代表该事项无结果，只是意识形态薄弱 |

#### 大格局识别（优先级最高）

**格局保护机制**（最关键，决定凶限能否全身而退）:

| 格局名称 | 构成条件 | 取象 | 重要性 |
|---------|--------|------|--------|
| **双禄夹目标宫** | 目标宫位左右各有一颗禄星 | 护佑机制——即使大限凶化，该宫有保底，不易彻底崩盘 | ⭐⭐⭐⭐⭐ |
| **禄存在目标宫** | 禄存落在目标宫位 | 基本盘保底——被羊陀夹但委屈求全中有守 | ⭐⭐⭐⭐ |
| **天府守财库** | 天府在财库宫位 | 财库有制煞能力——空劫入宫须特别注意 | ⭐⭐⭐⭐ |
| **护佑格局完整** | 整体看盘所有的保护机制 | 凶限虽艰，仍能全身而退；护佑格局破损→可能彻底崩盘 | ⭐⭐⭐⭐⭐ |

**关键大格局表**:

| 格局名称 | 构成条件 | 取象 |
|---------|--------|------|
| **府相朝元** | 命宫（或对宫）武贪+三合天府+天相 | 综合管理能力强，财经稳健，有格局 |
| **紫府朝垣** | 紫微+天府在三合四正 | 权贵格局，领导力强 |
| **禄马交驰** | 禄存与天马同宫或对宫 | 财运动态来财，奔波中有财 |
| **阳梁昌禄** | 太阳+天梁+文昌+化禄四星聚 | 科甲名声格，适合学术教育公职 |

#### 识别步骤

1. 查看事项的主看宫位和兼看宫位
2. 确认这些宫位是否有空宫（若空则按空宫规则处理）
3. 扫描原局是否有对这些宫位有利的保护格局
4. 特别关注双禄夹、禄存等护佑机制是否完整
5. 记录识别结果

### 步骤三：事项宫位四维合参

#### 目标
得出事项宫位的先天格局定性（强旺顺遂/中等有阻/虚浮困难/凶危高风险）

#### 四维合参标准流程

**按照本→对→合→临顺序分析**：

| 维度 | 分析内容 | 判断要点 |
|------|--------|---------|
| **本宫** | 主星旺弱 + 所有星曜（原局四化/六吉/六煞） | 该宫位本身的能量等级有多高 |
| **对宫** | 对宫的星曜状况和投射方向 | 是加强本宫？还是制约本宫？ |
| **合宫** | 三合宫的支撑状态 | 是强力后援？还是侧翼受压？ |
| **临宫** | 左右邻宫夹制情况 | 左右力量是否均衡，是激励还是压制 |
| **命宫调节** | 用命宫强弱作为基准调节 | 命强能驾驭凶象，命弱凶象穿透 |

#### 详细分析步骤

<details>
<summary><strong>点击展开详细步骤</strong></summary>

##### 本宫分析

1. 查看主看宫位的主星是什么
2. 评估主星在该宫的庙陷旺衰
3. 统计该宫所有星曜：
   - 四化星（禄权科忌）
   - 六吉（左辅右弼文昌文曲天魁天钺）
   - 六煞（羊陀火铃空劫）
   - 丙丁级（红鸾天喜咸池天姚破碎天虚）
4. 综合得出本宫的能量等级

##### 对宫分析

1. 查看对宫的星曜情况
2. 判断对宫与本宫的关系：
   - 加强本宫（对宫有吉星）
   - 制约本宫（对宫有煞星）
   - 中立（对宫星曜一般）
3. 记录对宫的投射方向

##### 合宫分析

1. 查看三合宫（与主宫成120°的两个宫位）
2. 评估三合宫的支撑程度：
   - 强力支撑（三合有吉星）
   - 侧翼受压（三合有煞星）
   - 一般（三合星曜平性）
3. 记录支撑情况

##### 临宫分析

1. 查看左右邻宫是否成立夹制（左右各有星）
2. 如成立，评估两侧力量是否均衡
3. 判断夹制是激励还是压制
4. 记录夹制结果

##### 命宫调节

1. 查看命宫的强弱定性（从命主性格定性中）
2. 用命宫强弱调节宫位的表现：
   - **命宫强** → 即使宫位有凶象也能驾驭
   - **命宫弱** → 宫位的煞象容易直接穿透

</details>

#### 四维合参结果定性

根据四维合参结果，得出宫位的能级定性：

| 定性 | 特征 | 判断标准 |
|------|------|---------|
| **强旺顺遂** | ✅ 星曜庙旺，格局护佑完整 | 主星庙旺，吉星多，对宫加强，保护机制完整 |
| **中等有阻** | ⚖️ 吉凶混杂，有挑战但可克服 | 主星平性，吉凶混杂，有支撑但也有压制 |
| **虚浮困难** | ⚠️ 星曜失陷，关键格局缺失 | 主星失陷或空宫，关键格局缺失，支撑不足 |
| **凶危高风险** | 🚫 空劫冲破，无护佑机制 | 主星失陷+煞多，护佑格局破损，风险极高 |

## 第三阶段：行运分析

### 核心原则

> 行运定激活——大限宫干四化引动原局格局，决定这十年往哪个方向走

### 步骤一：建立三层宫位对照表

#### 固定宫位顺序

```
命 → 父母 → 福德 → 田宅 → 官禄 → 仆役 → 迁移 → 疾厄 → 财帛 → 子女 → 夫妻 → 兄弟
```

#### 建立三层对照

| 层级 | 宫位地支 | 宫位排列方法 |
|------|--------|-----------|
| **原局** | 原局命宫地支 | 从原局命宫地支出发，按固定顺序排列12宫 |
| **大限** | 大限命宫地支 | 从大限命宫地支出发，按固定顺序排列12宫 |
| **流年** | 流年命宫地支 | 从流年命宫地支出发，按固定顺序排列12宫 |

#### 对照表作用

通过三层对照表可以快速看到：
- 原局的某个宫位在大限中落在哪里
- 大限的某个宫位在流年中落在哪里
- 便于进行大限行运梳理和流年分析

---

### 步骤二：逐大限行运梳理

#### 核心逻辑

**大限宫干四化是行运分析的核心工具**：

| 情况 | 含义 | 影响 |
|------|------|------|
| **吉化引动原局吉格** | 禄/权/科入好宫或好星 | 这十年原局吉象充分显现，趋势向好 |
| **忌化引动原局煞宫** | 忌入煞星聚集或凶宫 | 这十年原局凶象被激发放大，需要谨慎 |
| **原局有护佑格局** | 双禄夹、天府等 | 凶限虽艰仍有底线，全身而退有可能 |
| **原局无护佑格局** | 无保护机制 | 凶限直接穿透，须提前预警和防范 |

#### 逐大限梳理步骤

1. **列出命主的所有大限**（从第一个大限到当下大限）

2. **对每个大限，分析以下内容**：

   | 分析项 | 方法 | 记录 |
   |--------|------|------|
   | **大限命宫落位** | 看大限命宫在原局三层对照表中落在哪个宫位 | 落在原局[宫位名] |
   | **大限宫干四化** | 读取大限命宫天干的四化（禄权科忌各落哪里） | 大限[天干]化禄入[宫位]等 |
   | **四化引动方向** | 吉化引动吉格还是忌化引动凶象 | 吉化激活[什么好格局] / 忌化激活[什么凶象] |
   | **事项宫位激活** | 该大限中，主看事项宫位在哪里、是否被激活 | 事项宫位[宫位名]处于[激活/沉寂]状态 |
   | **这十年的走向** | 综合以上判断这十年该事项的总体趋势 | [顺畅期/艰辛期/危机期/转机期] |

3. **对每个大限做简要记录**，格式如下：

   ```
   【第X大限（年龄XX-XX）】
   大限命宫地支：[地支]
   大限命宫落原局：[宫位]
   大限宫干四化：[禄权科忌落位]
   四化引动：[具体格局或凶象]
   事项宫位状态：[具体表现]
   该十年定性：[顺畅/艰辛/危机/转机]
   ```

#### 示例

```
【第一大限（0-10岁）】
大限命宫地支：子
大限命宫落原局：官禄宫
大限宫干四化：癸禄入财帛+化权入命宫
四化引动：禄权激活财运，命宫获权杖
事项宫位状态（求财为例）：财帛宫处于激活状态
该十年定性：顺畅期（财运顺利发展）

【第二大限（10-20岁）】
大限命宫地支：亥
...
```

### 步骤三：当下大限深析

#### 目标
对当下大限（命主现在所处的十年）做最详细的分析

#### 分析内容

| 内容 | 分析方法 | 输出 |
|------|--------|------|
| **大限命宫×原局命宫合参** | 看大限命宫落在原局哪处，结合原局命宫 | 定这十年命主的性格状态 |
| **大限四化完整落位** | 逐一分析四个四化各落在哪里 | 吉化激活什么，忌化激活什么 |
| **大限主看事项宫四维** | 对该大限中的事项宫位做四维合参 | 这十年该事项的被激活程度 |
| **大限护佑机制检验** | 原局的保护格局在此大限是否完整 | 凶限是否仍有底线 |
| **大限整体定性** | 综合以上判断这十年的性质 | 顺畅期/艰辛期/危机期/转机期 |

#### 大限定性标准

| 定性 | 特征 | 对应表现 |
|------|------|---------|
| **顺畅期** | 四化吉多，原局吉象充分激活 | 事项有利，推进顺利 |
| **艰辛期** | 四化凶多，但有护佑格局 | 有挑战，需谨慎，但有底线 |
| **危机期** | 四化凶多，护佑格局破损 | 高风险，需预警，需防范 |
| **转机期** | 从凶象向吉象转变的中间阶段 | 变化期，把握窗口很重要 |

## 第四阶段：流年引动

### 核心原则

> 流年定引动——流年是导火线，引动大限已定的方向在今年具体触发

### 三层合参

| 层级 | 分析内容 | 作用 |
|------|--------|------|
| **流年×大限命宫** | 定今年触发点 | 原局作隐性背景 |
| **流年四化落位** | 今年四化引动哪些宫位 | 具体的年度引动 |
| **流年主看事项宫** | 今年该事项的具体引动状态 | 事项推进的实际方向 |

### 流年与大限方向判断

#### 判断矩阵

| 流年方向 | 大限方向 | 强度 | 判断 | 建议 |
|---------|--------|------|------|------|
| **吉** | **吉** | ⬆️⬆️ | **最佳推进窗口** | ✅ 主动出击 |
| **吉** | **凶** | ⬆️↔️ | **部分化解** | ⚠️ 谨慎推进 |
| **凶** | **吉** | ↔️⬇️ | **有波折干扰** | ⏸️ 维持为主 |
| **凶** | **凶** | ⬇️⬇️ | **风险最高** | 🚫 预警+规避 |

#### 具体说明

- **吉◇吉** - 双重叠加，两个层级都在推助，是最好的推进时机
- **吉◇凶** - 流年有好象在一定程度上化解大限的凶象，但凶象仍在，需要谨慎
- **凶◇吉** - 大限底盘仍然稳定，今年有波折但不会彻底崩盘
- **凶◇凶** - 两个层级都在压制，风险最高，需要提前预警和防范

### 流年分析步骤

1. **确定流年四化** - 读取流年命宫天干的禄权科忌
2. **确定流年方向** - 吉化多还是忌化多，判断流年整体趋势
3. **对标大限方向** - 看流年与大限是否一致
4. **判断今年触发点** - 综合两者判断今年该事项的具体表现
5. **确定时间窗口** - 是推进窗口、蛰伏期还是风险期

### 流年输出内容

```
【流年引动】

流年四化落位：
  • [禄入X宫] [权入X宫] [科入X宫] [忌入X宫]

今年事项引动方向：
  • [推进/维持/退守/转变]

流年与大限方向：
  • 流年[吉/凶] ◇ 大限[吉/凶] 
  • [一致则力量叠加/相反则互相制衡/部分化解等]

时间窗口：
  • [推进窗口/蛰伏期/风险期]
```

## 📊 快速参考表

### 宫位强弱定性对照表

| 定性 | 本宫特征 | 对宫情况 | 合宫情况 | 临宫情况 |
|------|---------|---------|---------|---------|
| **强旺** | 主星庙旺，吉星多 | 加强本宫 | 强力支撑 | 无夹制或激励夹 |
| **中等** | 主星平性，吉凶混杂 | 一般或分半 | 一般支撑 | 均衡夹制 |
| **虚浮** | 主星失陷或空宫 | 制约本宫 | 侧翼受压 | 压制夹制 |
| **凶危** | 主星失陷，煞多吉少 | 严重制约 | 严重受压 | 严重压制 |

### 大限定性判断标准

| 定性 | 四化特征 | 格局保护 | 对事项的影响 |
|------|---------|---------|-----------|
| **顺畅** | 吉化多，忌化少 | 不重要 | 事项顺利推进 |
| **艰辛** | 忌化多，吉化少 | **必须有** | 有难度但有底线 |
| **危机** | 忌化多，吉化少 | **必须无** | 高风险，彻底崩盘 |
| **转机** | 从凶→吉的过渡 | 有最好 | 变化期，需把握 |

### 流年定性快速判断

```
吉化多 + 吉化入好宫  →  推进窗口 ✅
吉化多 + 吉化入坏宫  →  蛰伏期 ⏸️
忌化多 + 忌化入坏宫  →  风险期 🚫
忌化多 + 有好的大限支持  →  挑战期 ⚠️
```
