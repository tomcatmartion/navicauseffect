# SKILL：宫位原生能级评估

版本：V3.1

核心升级：格局库完整落地——84条格局按大吉/中吉/小吉/小凶/中凶/大凶六级分类，成格条件与引动条件完整收录；格局判定流程三步化。

定位：输入命盘数据，输出12宫原生旺弱能级评分与基调。结果作为全系统锚点，后续模块直接调用。

边界：本Skill只评定宫位原生能级，事项分析、大限流年引动均属独立模块。

 

## 核心原则：宫位旺弱决定一切

全系统底层逻辑：宫位旺弱决定该宫所有星曜的呈现面貌与能量强弱。

 

**宫位亮度统一原则：**

第一步骨架评分定出宫位旺弱后，该宫所有星曜的亮度统一等于宫位亮度，不再看星曜自身的独立亮度标注。

| **宫位旺弱** | **该宫所有星曜的呈现方式**                     | **举例**                                                     |
| ------------ | ---------------------------------------------- | ------------------------------------------------------------ |
| 极旺/旺宫    | 所有星曜正面发挥——吉星真吉，煞星也呈现正面特质 | 旺宫擎羊→坚决勇往直前；旺宫火星→爆发力强；旺宫文昌→才华真实展现 |
| 平宫         | 所有星曜中性呈现——吉不完整，煞不极端           | 平宫文昌→才华中等发挥；平宫火星→有冲劲但不极端               |
| 陷宫/极弱    | 所有星曜负面放大——吉星虚华，煞星逞凶           | 陷宫擎羊→莽撞冲动；陷宫火星→暴躁破坏；陷宫文昌→才华虚华      |
| 空宫         | 完全依赖外部投射，无本宫承载                   | 空宫一切依赖对宫三合夹宫的投射质量                           |

 

**空间投射亮度原则：**

对宫/三合宫/夹宫的星曜投射到本宫时，星曜亮度由该星曜所在宫的旺弱决定（不是本宫旺弱），再乘以空间衰减系数。

例：对宫旺宫有火星 → 火星在旺宫呈正面发挥（被制） → 按旺宫煞星分值 × 0.8 投射本宫，煞力相对轻；对宫陷宫有火星 → 火星逞凶 → 按陷宫煞星分值 × 0.8 投射，煞力较重。

 

**制煞/****逞凶的定位：**

制煞/逞凶是定性标注，不影响评分数值。评分只看吉星加分与煞星减分的数值计算；制煞/逞凶标注供后续分析模块解读星曜赋性时使用。

 

| **本宫状态** | **承载力** | **分值处理**                                                 |
| ------------ | ---------- | ------------------------------------------------------------ |
| 极旺/旺宫    | 强         | 吉星按旺档加分；煞星按旺档减分（数值小）；标注「制煞」       |
| 平宫         | 中         | 吉星按平档加分；煞星按平档减分                               |
| 陷宫/极弱    | 弱/无      | 吉星按陷档加分（虚华）；煞星按陷档减分（数值大）；标注「逞凶」 |
| 空宫         | 极弱       | 完全依赖外部投射，无本宫承载                                 |

## 星曜吉煞分类（V3.1统一）

所有星曜统一固定分值，不因宫位旺弱而放大或缩小。宫位旺弱只影响骨架基础分与天花板。

| **分类**  | **包含星曜**                             | **固定分值**    | **处理方式**                                    |
| --------- | ---------------------------------------- | --------------- | ----------------------------------------------- |
| 吉星      | 化禄、左辅、右弼、文昌、文曲、天魁、天钺 | 每颗 +0.5       | × 空间衰减系数后加入Current_Score               |
| 煞星      | 火星、铃星、地空、地劫、化忌、擎羊、陀罗 | 每颗 -0.5       | × 空间衰减系数后加入Current_Score               |
| 化权/化科 | 化权、化科                               | 不加减分        | 仅作格局成格条件（如三奇嘉会）使用              |
| 禄存      | 禄存                                     | +0.5 / +0.3 / +0.1 | 第五步单独处理：旺宫+0.5，平宫+0.3，陷宫或空宫+0.1 |

擎羊/陀罗不再归禄存随行结构，统一按煞星-0.5计算。禄存单独在第五步处理。

化权/化科不参与加减分，只用于格局成格条件判断。

## 空间衰减系数

**对宫与三合宫（固定系数）：**

| **空间关系**       | **衰减系数** |
| ------------------ | ------------ |
| 本宫               | × 1.0        |
| 对宫               | × 0.8        |
| 三合宫（各自独立） | × 0.7        |

 

**夹宫动态系数（本宫旺弱 × 夹宫旺弱）：**

夹宫成立前置检查（必须先完成，不成立则不按夹宫计算）：

夹宫必须由成对星曜从左右两侧各一颗形成，以下为有效夹宫组合：

| 夹宫类型 | 构成条件 | 效果 |
|---|---|---|
| 昌曲夹 | 文昌在本宫左侧一宫＋文曲在本宫右侧一宫（或反之） | 吉夹 |
| 魁钺夹 | 天魁在本宫左侧一宫＋天钺在本宫右侧一宫（或反之） | 吉夹 |
| 左右夹 | 左辅在本宫左侧一宫＋右弼在本宫右侧一宫（或反之） | 吉夹 |
| 羊陀夹 | 擎羊在本宫左侧一宫＋陀罗在本宫右侧一宫（或反之） | 煞夹，羊陀夹忌为致命凶格 |
| 空劫夹 | 地空在本宫左侧一宫＋地劫在本宫右侧一宫（或反之） | 煞夹 |
| 火铃夹 | 火星在本宫左侧一宫＋铃星在本宫右侧一宫（或反之） | 煞夹 |
| 紫府夹 | 紫微在本宫左侧一宫＋天府在本宫右侧一宫（或反之） | 吉夹 |

不成立情况（严禁误判为夹宫）：
- 文昌文曲同在一宫 → 只影响本宫，不构成夹
- 天魁天钺同在一宫 → 只影响本宫，不构成夹
- 左辅右弼同在一宫 → 只影响本宫，不构成夹
- 单星在一侧无对应 → 不构成夹，只作普通单侧投射计算
- 非成对组合的两颗星分处两侧 → 不构成夹，各自独立按普通投射计算

成立后按以下系数计算：

| **本宫 \ 夹宫** | **夹宫旺** | **夹宫平** | **夹宫弱** |
|---|---|---|---|
| 本宫旺 | × 0.9 | × 0.5 | × 0.2 |
| 本宫平 | × 0.6 | × 0.4 | × 0.2 |
| 本宫弱 | × 0.3 | × 0.1 | × 0 |

 

**禄存专项规则（第五步）：**

擎羊/陀罗已在第四步作为煞星按-0.5计算完毕。禄存在第五步单独处理，只看禄存所在宫的宫位旺弱。

| **宫位旺弱**   | **禄存加减分** | **定性说明**   |
| -------------- | -------------- | -------------- |
| 旺宫/极旺      | +0.5           | 乘旺真实发挥   |
| 平宫           | +0.3           | 中性保底加分   |
| 陷宫/极弱/空宫 | +0.1           | 委曲虚弱仍有守 |

 

## 六步评分流程

严格按顺序执行，不得跳步。加分阶段完成后才判断宫位旺弱，再执行减分。

 

### 第一步：骨架基础分

从骨架映射库读取本宫主星状态，锁定初始基础分与天花板。Current_Score = 初始基础分。

同时记录主星制煞能力等级：强制煞（紫微/天府）、中制煞（太阳旺/武曲旺/廉贞旺/七杀旺/破军旺/贪狼旺）、弱制煞（天机/太阴/天同/天梁）。

 

### 第二步：加分阶段（顺序执行①～⑦）

**①** **三方四正吉星加分**

扫描本宫+对宫+两个三合宫+两侧夹宫，凡见以下吉星，按空间衰减系数加分：

- 化禄、左辅、右弼、文昌、文曲、天魁、天钺：各 +0.5 × 空间衰减系数
- 化权/化科：只记录位置，不加分，仅供格局判断使用

**②** **命主生年四化加分／减分**

命主生年天干四化，落入哪宫则对该宫直接加减分：
- 化禄落位宫：+0.5（本宫全值，三方四正按空间衰减系数计算）
- 化忌落位宫：-0.5（本宫全值，三方四正按空间衰减系数计算）
- 化权／化科：只记录落位，不加减分，仅供格局判断

**③** **命主太岁宫遁干四化加分／减分**

命主生年地支 = 太岁宫，查五虎遁干直查表得遁干，再查四化索引表得遁干四化，落入哪宫则对该宫加减分：
- 遁干化禄落位宫：+0.5（本宫全值，三方四正按空间衰减系数计算）
- 遁干化忌落位宫：-0.5（本宫全值，三方四正按空间衰减系数计算）
- 遁干化权／化科：只记录落位，不加减分，仅供格局判断

> 说明：生年四化与遁干四化地位等同，均须完整计算，不得只计算其中一套。

**④** **父亲生年四化——化禄加分／化忌减分**

父亲生年干四化，化禄落入哪宫则对该宫加分+0.5×0.9折算，化忌落入哪宫则减分-0.5×0.9折算。化权/化科只记录，不加减分。

**⑤** **父亲太岁宫遁干四化——化禄加分／化忌减分**

父亲生年地支 = 太岁宫，查五虎遁干直查表得遁干，化禄落位加分+0.5×0.9折算，化忌落位减分-0.5×0.9折算。

**⑥** **母亲生年四化——化禄加分／化忌减分**

母亲生年干四化，化禄落位加分+0.5×0.9折算，化忌落位减分-0.5×0.9折算。化权/化科只记录，不加减分。

**⑦** **母亲太岁宫遁干四化——化禄加分／化忌减分**

母亲生年地支 = 太岁宫，查五虎遁干直查表得遁干，化禄落位加分+0.5×0.9折算，化忌落位减分-0.5×0.9折算。

 

**⑦** **吉格加分**

查格局库，成格条件与引动条件均满足则触发。

**格局系数作用范围规则（重要）：**
- 格局系数只作用于「触发该格局的核心宫位」，不得套用到全盘所有宫位
- 触发格局的核心宫位：格局构成星曜所在宫位 + 引动条件所在宫位
- 非触发格局的其他宫位：保持原分值，不受该格局系数影响
- 同一宫位可能触发多个格局，取最高格局系数应用，不叠加

**格局系数对照：**

| 格局级别 | 系数 | 作用范围 |
|---|---|---|
| 大吉格局 | ×1.5 | 仅触发该格局的核心宫位 |
| 中吉格局 | ×1.3 | 仅触发该格局的核心宫位 |
| 小吉格局 | ×1.0（只标注，不加分） | 标注即可，不修改分值 |

三奇嘉会格触发条件：同一来源（命造/父亲/母亲三者之一）的化禄+化权+化科同时会于三方四正，或命宫+两侧夹宫凑齐同一来源的科权禄，则成格（中吉×1.3）。四化来源必须严格隔离，不同来源的科权禄不得混算。格局系数只作用于三奇嘉会的核心宫位，不扩散至全盘。

 

### 第三步：加分完毕——判断宫位旺弱

以加分后的Current_Score落入的分值区间，重新定性本宫旺弱等级，作为后续减分幅度的基准。

| **加分后Current_Score** | **重新定性**         |
| ----------------------- | -------------------- |
| ≥ 7.5                   | 旺（强承载）         |
| 6.0 ～ 7.4              | 旺偏磨炼（中强承载） |
| 4.5 ～ 5.9              | 平（中承载）         |
| 3.0 ～ 4.4              | 虚浮（弱承载）       |
| < 3.0                   | 陷（无承载）         |

重新定性后：旺宫可触发制煞为用；陷宫遇吉须标注虚华；平宫按普通处理。

 

### 第四步：减分阶段（顺序执行⑧～⑩）

**⑧** **三方四正煞星减分**

扫描本宫+对宫+两个三合宫+两侧夹宫，凡见以下煞星，按空间衰减系数减分：

•    擎羊、陀罗：归禄存随行结构，本步不处理，留至第六步

•    火星、铃星、地空、地劫、化忌：此步处理，按第三步定性的宫位旺弱决定扣分幅度

 

**⑨** **父母煞星减分**

父亲生年四化化忌 + 父亲太岁宫遁干化忌已在加分阶段④⑤中处理完毕（×0.9折算）。
母亲同上，已在⑥⑦中处理完毕。
本步骤确认无遗漏，不重复计算。

 

**⑩** **凶格减分**

查格局库，成格则按级别应用倍率（中凶×0.7，大凶×0.5，小凶×1.0只标注）。

 

### 第五步：禄存判断

擎羊/陀罗已在第四步⑧作为煞星处理完毕（各-0.5×衰减系数）。本步只处理禄存，查第三步定性后的宫位旺弱：旺+0.5，平/旺偏磨炼+0.3，虚浮/凶危+0.1。

 

### 第六步：天花板截断与基调定论

Final_Score = min(Current_Score, 初始阶层天花板)

 

| **Final_Score**  | **原生基调**   | **核心特征**                   |
| ---------------- | -------------- | ------------------------------ |
| 7.5 ～ 8.5       | 🟢 实旺（沃土） | 承载力极强，制煞为用可充分发挥 |
| 6.0 ～ 7.4       | 🟢 实旺偏磨炼   | 整体强旺，有小幅摩擦           |

| 1.5 ～ 2.9       | 🔴 凶危（雷区） | 忌煞交冲，化禄常为诱饵         |

**强制绝败触发（无论分值直接判定⚫****）：**

•    忌星共振4颗以上

•    陷宫主星 + 火铃 + 化忌三重同宫

•    化禄与化忌同宫且主星陷——禄被忌锁无缓冲

| 平宫临界     | 第三步定性平 + 减分后接近虚浮   | 行运吉则向吉象转化，行运忌则向凶象滑落     |

## 执行总流程

1. 确认前置数据就绪
2. 逐宫第一步：骨架库锁定初始分/天花板/制煞等级
3. 逐宫第二步：加分阶段——①三方四正吉星②命主太岁宫遁干化禄③父亲生年化禄④父亲太岁宫遁干化禄⑤母亲生年化禄⑥母亲太岁宫遁干化禄⑦吉格
4. 逐宫第三步：加分完毕，以当前分值重新定性宫位旺弱
5. 逐宫第四步：减分阶段——⑧三方四正煞星⑨父母化忌⑩凶格
6. 逐宫第五步：禄存随行结构判断（旺宫加分/陷宫减分）
7. 逐宫第三步：制煞判定 + 双煞/三煞预警 + 格局库三步检查
8. 逐宫第六步：天花板截断 → Final_Score → 基调定论
9. 强制绝败检查
10. 临界状态标注（吉煞比0.7～1.2区间）
11. 12宫基调输出，命宫/官禄/财帛优先
12. 全系统基调锁定，移交后续模块

 

## 输出禁令

•    严禁跳过第二步直接算分——必须先汇总有效吉煞权重

•    严禁擎羊/陀罗计入六煞数量——归禄存随行结构

•    严禁在未确认本宫主星旺弱之前对任何注入能量做吉凶判断

•    严禁陷宫遇吉做真吉论——陷地虚华必须标注

•    严禁平宫/陷宫做制煞为用论

•    严禁机月同梁系做制煞为用论

•    严禁陷禄做制煞为用论——虚火

•    严禁本宫三煞以上仍做制煞为用论——强制顶级凶格

•    严禁格局检查只看成格条件不看引动条件——两步均须满足

•    严禁忽略天花板截断

•    严禁将大限流年四化纳入本Skill计算

•    严禁跳步执行——六步顺序不可颠倒，每步结果是下一步的输入：第三步旺弱定性依赖第二步加分完毕；第四步减分幅度依赖第三步定性；第五步禄存判断依赖第四步完成；跳步必然导致旺弱基准错误或减分错位

•    严禁在后续模块里重新评估宫位旺弱——本Skill输出全系统锁定
