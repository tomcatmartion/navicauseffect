### 1. 命宫（太极点）

**【检索关键词】**
性格定性 | 天赋属性 | 命主表现 | 核心轴心 | 身宫后天 | 性格才智 | 发展程度 

**【基础信息】**

| 项目 | 内容 |
|------|------|
| 别称 | 太极点 |
| 主要意义 | 整个命盘核心，代表本人，一生命运及周围人事物的枢纽 |
| 主管时期 | 少年至中年（40岁前）的贫富情况影响最大 |
|          |                                                    |

**【可看事项】**

- 容貌
- 性格
- 才能
- 思想意识
- 表面性格

**【具体主事】**

- 生命
- 本质
- 性情
- 人品
- 才能
- 发展
- 人生定位
- 人生层次

**【意识形态】**（核心理解）

命宫代表命主「是什么样的人」——性格、才智、天赋属性
命宫旺→主见强，有能力驾驭三方吉凶；命宫弱→易受外界影响，三方的煞容易直接穿透

**【重点星曜】**

- 官禄主星（紫微/天府/廉贞/天相/武曲/太阳）坐命→事业心强，处理事务能力强
- 机月同梁坐命→侧重关注人际关系以及对事理的探究；但太岁宫若为官禄主则内核仍有强烈事业心
- 命宫见化忌→终身携带执念底色，全程带入所有分析

**【分析要点】**

**内部因素**: 命宫本宫星曜的力量组合、星的旺衰、落宫的生旺衰绝

**外部因素**: 三方四正及左右邻宫的星曜扶抑情况

**强弱程度判断**

| 级别 | 判断标准 |
|------|----------|
| 最强 | 有好格局；庙旺正星得宫生，落宫长生帝旺，副星均吉无煞 |
| 强 | 主星庙旺，吉多煞少；或无主星但有吉无煞对宫好 |
| 一般 | 主星平性，吉凶混杂；或主星失陷无煞有吉 |
| 弱 | 主星平性吉凶混杂；或失陷吉凶混杂 |
| 最弱 | 成贫贱格局；失陷无吉恶煞众多 |
| 无正星 | 借对宫所有星曜过来与原宫副星一起论，吉凶打折扣（约50%） |

**【常见误用纠正】**

❌ 命宫旺不等于成就高

✅ 成就需要先天素质+后天环境配合，不能只看命宫论成就

❌ 命身宫只代表属性，不代表宿命

✅ 切忌搞宿命论

**【取象举例】**

- 命宫弱但迁移宫强→外表柔弱但出门如鱼得水，靠外部人脉与环境成事
- 命宫强但三方煞重→有能力但阻碍多，越挫越勇

**【应用场景速查表】**

| 事项类型 | 应用方式 | 主要宫位 | 备注 |
|--------|--------|--------|------|
| 性格分析 | 命宫必看 | 命宫 | 所有事项分析的前提 |
| 家庭关系 | 配合迁移宫和六亲 | 命宫 | 命宫强弱影响所有关系 |
| 事业发展 | 结合官禄财帛田宅 | 命宫 | 命宫弱需补充格局分析 |
| 婚姻感情 | 配合夫妻宫 | 命宫 | 命强能驾驭感情 |

**【与其他宫位的关系】**

- **迁移宫**: 命宫主内部，迁移宫主外部；命宫无破迁移宫吉旺适合做对外相关事宜，命宫凶迁移宫旺则需注意被人利用。
- **福德宫**: 福德宫吉旺满足感强；福德宫凶则欲望大，不满足
- **疾厄宫**: 命强者疾厄有凶也能扛；命弱则煞直接穿透
- **身宫**: 身宫是命宫里层，是当遇到问题时所表现的性格特征，第三个大限后凸显
- **太岁宫：**太岁宫是人的内核源动力，是行为的推动源

**【禁令规则】**（调用此宫必知）

⛔ 严禁跳过命主性格定性直接分析事项——命主性格是所有事项分析的前提

⛔ 严禁脱离命宫强弱解读任何事项——命宫是所有事项的统领轴心

⛔ 严禁单宫论断——任何宫位判断必须本对合临四维合参

### 2. 兄弟宫

**【检索关键词】**
兄弟姐妹 | 手足关系 | 母亲位 | 亲情 | 手足亲厚 | 堂表亲 

**【基础信息】**

| 项目 | 内容 |
|------|------|
| 别称 | 无 |
| 主要意义 | 兄弟姐妹的容貌性情才能、兄弟间关系好坏及助益 |
| 主管时期 |  |
| 六亲对应 | 母亲、配偶的父亲 |

**【可看事项】**

- 兄弟姐妹
- 母亲
- 亲密朋友
- 事业伙伴
- 成就
- 费用支出

**【具体主事】**

- 兄弟情义

**【意识形态】**（核心理解）

兄弟宫代表对待兄弟姐妹的意识形态和亲缘关系
兄弟宫吉→与兄弟姐妹亲厚；兄弟宫有问题→疏离或冲突矛盾
天机星好的人（无论在哪宫）→对兄弟姐妹更包容，即使兄弟宫有问题也有化解

**【重点星曜】**

- 天机坐兄弟→最包容兄弟姐妹，即使有凶象也能化解
- 吉星多（左右禄权科）→兄弟姐妹众多且关系好，互相扶持
- 空劫入兄弟→兄弟姐妹缘分浅，或数量少
- 化忌入兄弟→容易与兄弟姐妹产生矛盾和冲突

**【分析要点】**

**内部因素**: 兄弟宫本宫星曜的旺衰与星性

**外部因素**: 三方四正及天机星的影响；吉星多煞少则关系好有助益

**【常见误用纠正】**

❌ 兄弟宫≠现金流宫

✅ 现金流看福德宫；

❌ 无法判断有几个兄弟姐妹

✅ 受环境影响太大，现代独生子女普遍，不能作为判断依据

❌ 兄弟宫化忌=与兄弟姐妹关系必定差

✅ 须带入对方太岁入卦辨别，忌的程度决定冲突大小

**【取象举例】**

- 兄弟宫吉星多→出生在大家庭，兄弟姐妹众多且关系融洽
- 兄弟宫煞多+化忌→与某位兄弟姐妹关系紧张，容易产生纠纷
- 疾厄宫弱+兄弟宫凶+忌煞交冲→进一步降低抵抗力

**【与其他宫位的关系】**

- **父母宫**: 兄弟宫与父母宫构成原生家庭基础，属于命宫的夹宫，是人身边环境，对人一生影响大
- **命宫**: 命宫左邻为兄弟，命宫强则能驾驭兄弟不和
- **疾厄宫**: 兄弟宫凶会增加疾厄宫的问题程度

**【禁令规则】**（调用此宫必知）

⛔ 严禁用兄弟宫看朋友——朋友看迁移宫或奴仆宫

⛔ 严禁用兄弟宫看现金流——现金流看福德宫

⛔ 严禁忽视天机星的包容能力——天机好可以化解兄弟宫的问题

### 3. 夫妻宫

**【检索关键词】**
感情 | 伴侣 | 婚姻 | 择偶 | 桃花 | 亲密关系 | 异性缘 | 婚姻线

**【基础信息】**

| 项目 | 内容 |
|------|------|
| 别称 | 无 |
| 主要意义 | 配偶容貌、性情、才能、婚姻情况及夫妻关系 |
| 主管时期 | 成年后至终身关系 |
|          |                                          |

**【可看事项】**

- 配偶
- 恋爱对象
- 异性缘
- 婚姻质量

**【具体主事】**

- 异性缘
- 恋爱
- 婚姻
- 感情
- 夫妻生活

**【意识形态】**（核心理解）

夫妻宫代表对伴侣的意识形态，即「喜欢什么类型的伴侣」以及「如何处理亲密关系」
不是预测「未来伴侣是什么样的人」——而是命主自身的择偶倾向与关系处理模式
桃花旺→异性缘强，感情吸引力大；桃花弱→异性缘薄，感情难得

**【重点星曜】**

- 甲级桃花星（太阴/太阳/廉贞/贪狼）入夫妻→感情吸引力强，桃花旺
- 丙级桃花星（红鸾/天喜/咸池/天姚）入夫妻→增加感情氛围，但可能流于表面
- 化禄入夫妻→感情顺遂，夫妻恩爱；化忌入夫妻→感情有阻滞或执念
- 夫妻宫见煞忌交冲→感情关系有较大波折

**【分析要点】**

**内部因素**: 夫妻宫本宫星曜的庙陷旺衰与桃花特性

**外部因素**: 三方四正的吉凶煞；对宫（奴仆宫）的影响；命身宫的强弱

**强弱程度判断**

| 级别 | 判断标准 |
|------|----------|
| 配偶好 | 夫妻宫星曜庙旺吉多→配偶容貌性情才能都好 |
| 婚姻美 | 夫妻宫强于命宫→可得益婚姻，两人和睦 |
| 有波折 | 夫妻宫吉凶混杂→感情有起伏但可接受 |
| 婚姻不美 | 夫妻宫弱→婚姻质量不如意 |
| 刑克 | 夫妻宫不好+三方见煞+日月失陷→有刑克风险 |
| 配偶差 | 夫妻宫失陷煞多→配偶条件差或个性不合 |

**【常见误用纠正】**

❌ 夫妻宫预测「伴侣是什么样的人」

✅ 根据命主喜欢的类型牵强附会是错的；夫妻宫是自己的意识形态，不是伴侣特征

❌ 晚婚象=必须晚婚

✅ 看结婚需要更多吉象聚合，现代婚姻要件多，不是定数

❌ 无主星或空星=晚婚一定成立

✅ 需结合福德/迁移/命身综合判断

**【取象举例】**

- 紫微破军化禄在夫妻宫+福德宫贪狼化忌→不是晚婚，而是对伴侣要求高，不想将就
- 夫妻宫有桃花星+正气吉星→自由恋爱感情好；桃花星多+煞→感情纠纷多

**【应用场景速查表】**

| 事项类型 | 应用方式 | 主要宫位 | 备注 |
|--------|--------|--------|------|
| 婚姻质量 | 夫妻宫+命宫 | 夫妻宫 | 判断婚姻是否幸福 |
| 配偶类型 | 夫妻宫三方 | 夫妻宫 | 推断择偶倾向 |
| 感情波折 | 夫妻宫+三方四正 | 夫妻宫 | 判断感情起伏 |
| 刑克风险 | 日月+夫妻宫 | 夫妻宫 | 男看太阴失陷丧妻，女看太阳失陷丧夫 |

**【与其他宫位的关系】**

- **命宫**: 夫妻宫强于命宫→婚姻能为人生加分；反之则拖累
- **福德宫**: 福德宫好→伴侣也会因此受益，夫妻关系更和谐
- **财帛宫**: 夫妻宫好+财帛好→家庭既幸福又富足

**【禁令规则】**（调用此宫必知）

⛔ 严禁用夫妻宫直接预测伴侣长相性格——这是牵强附会

⛔ 严禁忽视福德/迁移对婚姻的影响

⛔ 严禁脱离时间维度判断→必须看大限流年引动才能判断感情结果

### 4. 子女宫

**【检索关键词】**
生育 | 子女 | 生殖 | 有无小孩 | 怀孕 | 多胞胎 | 育儿 

**【基础信息】**

| 项目 | 内容 |
|------|------|
| 别称 | 无 |
| 主要意义 | 子女的人数、容貌、性情、才能及与自己的感情关系 |
| 主管时期 | 影响子女养育和家庭幸福 |
| 六亲对应 | 第二配偶、配偶的兄弟姐妹 |

**【可看事项】**

- 子女
- 学生弟子
- 晚辈
- 长子
- 生育能力

**【具体主事】**

- 子女
- 生育

**【意识形态】**（核心理解）

子女宫代表对子女的意识形态和亲缘关系
子女数量判断：子女宫吉星多→子女多且好；煞多→子女少或不肖
禄存/化禄入子女宫流年→怀孕的象；左辅右弼成对→容易多胎

**【重点星曜】**

- 禄存入子女宫空宫→舍不得花钱在子女上
- 禄存/化禄入子女宫流年→怀孕的象
- 左辅/右弼入子女宫→主观上愿意有多个子女；双胎父母之一子女宫大概率有成对左右
- 天同星好→跟小孩子沟通更顺畅（幼教/育儿方面加分）
- 空劫入子女→淡化对子女的重视，或无子女

**【分析要点】**

**内部因素**: 子女宫本宫星曜的旺衰与特性

**外部因素**: 三方四正的吉凶；禄存是否入子女宫

**强弱程度判断**

| 级别 | 判断标准 |
|------|----------|
| 子女多好 | 子女宫吉星多→子女众多，容貌性情才能都好，关系亲厚 |
| 子女好 | 主要吉星→子女少但优秀，聪慧孝顺 |
| 吉凶混 | 子女有好有差，教育需用心 |
| 子女少 | 空劫入子女→子女缘浅，难以成多 |
| 子女差 | 煞星多→子女不肖或不得力 |
| 无子女 | 化忌重+无吉→可能无子或断绝 |

**【常见误用纠正】**

❌ 子女宫空宫=没有孩子

✅ 空宫看格局，许多空宫命也有孩子

❌ 情欲滥情看子女宫

✅ 情欲滥情看桃花星（红鸾/天喜）

❌ 当老师必须子女宫好

✅ 热爱教育看命身宫格局，子女宫好只是加分项

**【取象举例】**

- 有孩子的可拿自己盘对照——怀孕流年子女宫应见禄
- 双胞胎父母子女宫大概率见成对左右
- 子女宫化忌+煞多→与子女的沟通存在问题

**【与其他宫位的关系】**

- **田宅宫**: 田宅不好难得好子孙（阳宅主要决定子孙好坏，阴宅主要决定子孙性格及地位）
- **福德宫**: 福德宫好→养育子女更从容，经济充裕
- **财帛宫**: 财帛不足→子女教育资源受限
- **夫妻宫**: 夫妻关系影响子女成长环境

**【禁令规则】**（调用此宫必知）

⛔ 严禁以子女宫化忌判断绝对无子——需看整体格局

⛔ 严禁用子女宫看情欲问题——情欲看桃花星不是子女宫

⛔ 严禁怀孕求子分析——此事项拒绝分析，直接告知超出范围

### 5. 财帛宫

**【检索关键词】**
来财方式 | 收入 | 报酬 | 利润观念 | 经营理念 | 薄利多销 | 求财

**【基础信息】**

| 项目 | 内容 |
|------|------|
| 别称 | 无 |
| 主要意义 | 一生经济活动、理财能力、财运发展、收入高低、赚钱方式 |
| 主管时期 | 直接影响一生经济地位和生活质量 |
| 六亲对应 | 孙辈、子女的朋友 |

**【可看事项】**

- 财运好坏
- 赚什么行业的钱
- 正财还是偏财
- 物质享受是否稳定

**【具体主事】**

- 财运
- 生意
- 投资
- 赚钱
- 现金
- 花钱
- 损财
- 资金周转

**【意识形态】**（核心理解）

财帛宫代表「如何对待来钱的方式」，官禄宫代表「如何做事」
先做事后来财：官禄宫象先行，财帛宫为后
官禄宫弱+财帛宫旺→想挣大钱但工作心态不扎实→制造业品控差；有权者可能谋私

**【重点星曜】**

- 武曲/天府/太阴（财星）入财帛宫→财利特点直接发挥，需合参财星星性
- 化禄入财帛→来财顺畅；化忌入财帛→来财有阻滞，严重时有损失

| 事业好 | 主星平性但吉多→事业稳定，发展顺利 |
| 事业一般 | 吉凶混杂→事业有起伏，需要努力 |

**【常见误用纠正】**

❌ 化忌在官禄=没有事业

✅ 有忌不一定有凶象，忌煞交冲才有凶象

❌ 同一件事不同行业看同一宫

✅ 贸易看迁移，纯投资看福德，房产投资看田宅；不同行业看不同宫

**【取象举例】**

- 快消品贸易关键在迁移宫（货品快速流动进销）
- 制造业关键在官禄宫（品控与生产理念）
- 官禄宫弱+财帛宫旺→想挣大钱但处理工作心态不扎实→制造业品控问题；有权力者可能利用手中权力谋私

**【与其他宫位的关系】**

- **迁移宫**: 迁移好助力事业发展；迁移差则事业推进困难
- **命宫**: 命宫强→能驾驭官禄凶象；命宫弱→官禄凶象直接显现

⛔ 严禁脱离命宫看官禄宫

**【意识形态】**（核心理解）

田宅宫是命主对置产的意识形态、居家环境、固定资产投资、资金链安全的宫位
田宅宫是藏财之所，是紫微斗数最重要的财宫之一
田宅宫旺→置产意识强，只要条件允许就会置业；田宅宫弱→缺乏置产意识，即便得到祖产也难充分发挥
财富等级：田宅宫比财帛宫更重要（能挣≠能存）

**【重点星曜】**

- 空劫入田宅→资金链有漏洞风险

**【分析要点】**

**内部因素**: 田宅宫本宫星曜的庙陷旺衰与星性

**外部因素**: 三方四正的吉凶；禄存/左右的辅助；对宫（官禄宫）的影响

**强弱程度判断**

| 级别 | 判断标准 |
|------|----------|
| 置产旺 | 太阴/天府（田宅主）入田宅宫→最适合，置产意识最强；禄存/左辅/右弼入田宅→有利置产 |
| 财库好 | 日月照壁格入田宅宫→非常好；禄存入→存钱能力强 |
| 一般 | 田宅宫吉凶混杂→置产意识一般，存钱能力中等 |
| 资金链弱 | 空劫入田宅→资金链有漏洞风险 |
| 置产难 | 破军巨门入田宅→先卖后买，容易换产 |

**【常见误用纠正】**

❌ 田宅宫旺=一定有很多房产

✅ 置产意识强+条件允许才能实现，用过程成因角度看

❌ 房地产中介看田宅宫

✅ 房地产中介是销售本质，重点是迁移宫

❌ 看房屋风水格局用田宅宫

✅ 需方位太岁入卦，不能单看田宅宫

**【取象举例】**

- 田宅宫破了的做生意的人基本都是大问题——固定资产抵押资金链断裂
- 同工不同酬问题有时与田宅宫有关——能挣的钱守不住
- 田宅宫与子孙：田宅不好难得好子孙（阳宅主要决定子孙好坏，阴宅主要决定子孙性格及地位）

**【应用场景速查表】**

| 事项类型 | 应用方式 | 主要宫位 | 备注 |
|--------|--------|--------|------|
| 置业意愿 | 田宅宫本宫 | 田宅宫 | 是否想买房 |
| 置业时机 | 田宅宫+流年 | 田宅宫 | 什么时候买房顺利 |
| 资金链 | 田宅宫+财帛宫 | 田宅宫 | 长期资金链是否安全 |
| 租住运 | 田宅宫吉凶 | 田宅宫 | 租住环境质量 |

**【与其他宫位的关系】**

- **财帛宫**: 财帛是挣钱能力，田宅是存钱能力；财富等级看两宫合参
- **福德宫**: 福德宫好→生活舒适；田宅宫好→住房满意，两者结合更幸福
- **官禄宫**: 做实体生意→官禄+田宅两宫都重要
- **子女宫**: 田宅不好难得好子孙；阳宅影响子孙好坏

**【禁令规则】**（调用此宫必知）

⛔ 严禁用田宅宫旺就判断一定有很多房产——需要置产意识+条件+时机三者合一

⛔ 严禁用田宅宫看房地产中介——中介是销售，重点看迁移宫

### 11. 福德宫

- 享乐
- 饮食
- 娱乐
- 思想行为
- 品德
- 财源

**【意识形态】**（核心理解）

福德宫是命主对生活享受的理念与态度、对外来信息的感知、消费观念、动产支配方式的宫位
现金流是否充盈宽裕、短期理财、借出款项回收→福德宫是重点宫位
福德宫稳→更懂得享受生活，不操心；福德宫不稳→容易情绪不稳，容易操心
福德宫影响命主能否接收正面信息——福德宫有问题的人容易被忽悠、深信谬论

**【重点星曜】**

- 空劫入福德宫→淡化对金钱的重视，容易存不下钱
- 禄存入福德宫空宫→舍不得花钱，守财倾向强
- 情绪稳定与否→与福德宫有关，福德宫有问题容易情绪不稳定
- 天府天相贪狼太阴天同有吉星吉化，见红鸾天喜更甚→会更喜欢享受

**【分析要点】**

**内部因素**: 福德宫本宫星曜的旺衰与特性

**外部因素**: 三方四正的吉凶；对宫（疾厄宫）的影响；命身宫的强弱

**强弱程度判断**

| 级别 | 判断标准 |
|------|----------|
| 福德旺 | 福德宫吉星多→幸福安逸，情绪稳定，现金流充盈 |
| 福德好 | 化禄入福德→享受生活，能接收正面信息 |
| 福德一般 | 吉凶混杂→生活有起伏，情绪不太稳定 |
| 福德差 | 化忌入福德→容易操心，情绪不稳 |
| 现金流弱 | 空劫入福德→存不下钱，现金流紧张 |
| 福德破 | 禄存入福德空宫→舍不得花钱，守财倾向强；双忌入→情绪很差，容易被骗 |

**【常见误用纠正】**

❌ 福德宫=性格宫

✅ 性格看命身太岁；福德宫看享受和情绪稳定性

❌ 爱好不能看福德宫

✅ 受命身宫影响更大，福德宫是辅助

❌ 情绪问题全看福德宫

✅ 需宏观看盘，相关星曜有问题也会发生对应情绪问题

**【取象举例】**

- 女性福德宫好很重要——手头宽裕、情绪稳定，即使为家庭牺牲事业也能开心生活
- 福德宫空宫禄存→一个钱分两瓣花，极度节省
- 福德宫有问题的人容易被忽悠、深信谬论——信息感知能力差

**【应用场景速查表】**

| 事项类型 | 应用方式 | 主要宫位 | 备注 |
|--------|--------|--------|------|
| 生活幸福度 | 福德宫本宫 | 福德宫 | 是否幸福满足 |
| 现金流 | 福德宫+兄弟宫 | 福德宫 | 短期流动资金充足 |
| 情绪管理 | 福德宫吉凶 | 福德宫 | 情绪稳定程度 |
| 投资理财 | 福德宫+迁移 | 福德宫 | 理财眼光和判断力 |

**【与其他宫位的关系】**

- **命宫**: 命好福好→自然无往不利，事业顺遂，家庭幸福；命好福不好→注意精神享受缺陷；命不好福好→虽有精神享受但命运层次受限
- **财帛宫**: 纯投资（无劳力付出）→重点看福德宫
- **田宅宫**: 福德宫好→生活舒适；田宅宫好→住房满意；两者都好→生活完美
- **迁移宫**: 福德好的人出门也能享受生活，遇到好机会

**【禁令规则】**（调用此宫必知）

⛔ 严禁用福德宫看性格——性格看命身太岁

⛔ 严禁把爱好全归到福德宫——受命身宫影响更大

⛔ 严禁用福德宫单独解释情绪问题——需宏观看盘

✅ 上司看迁移宫

❌ 父母宫差=一定无法上学

**【取象举例】**

- 离异家庭两个孩子，一个跟父一个跟母，父母宫象差异极大；不是看父母一生命运，而是看这个孩子与父母的缘分深浅

**【应用场景速查表】**

| 父母状况 | 太岁入卦 | 父母宫 | 父母的实际命运 |

|          |              |          |                  |

**【与其他宫位的关系】**

- **兄弟宫**: 兄弟宫与父母宫构成原生家庭基础，对人一生影响大
- **命宫**: 父母宫是命宫的外部补充；命父强弱相参

- **疾厄宫**: 父母太岁入卦判断遗传影响

## 补充：四化星在各事项中的含义

**说明**：四化内容保持原有的通用定义，不按事项重新定义。大模型在分析时根据具体宫位定性自行摘取。

### 化禄通用含义

- **本质**：财利、顺畅、结果；具备制煞能力
- **正面**：欲望明确，追求方向清晰，驱动力强；顺畅喜悦圆融；遇煞可制化
- **负面**：「禄为诱饵」——顺畅时容易不设防，因兴奋导致大意
- **在各宫的具体表现**：由该宫的V3.0定性决定（旺宫充分，陷宫虚华）

### 化权通用含义

- **本质**：权柄、信心、企图心
- **正面**：强势主导，控制欲与主导欲强，企图心旺；提供突破现状的硬动能
- **负面**：化权有时有嘚瑟意味；权忌交冲则凡事须经艰辛过程
- **在各宫的具体表现**：由该宫的V3.0定性决定（旺宫精准有力，陷宫强行蛮干）

### 化科通用含义

- **本质**：才智、名声、条理
- **正面**：理性清晰，注重名声与形象管理，条理有序；有利名望积累与贵人认同
- **负面**：化科力量较禄权弱，须有禄或权配合才能充分发挥
- **在各宫的具体表现**：由该宫的V3.0定性决定（旺宫名实相副，陷宫虚名）

### 化忌通用含义

- **本质**：干扰、不顺、力道减弱、衰退、冲动
- **正面**：专注执着（旺宫化忌）；在某一方向形成深度聚焦
- **负面**：执念阻滞，是最重要的底层张力；**忌煞交冲才构成真正凶象**
- **在各宫的具体表现**：由该宫的V3.0定性决定（旺宫专注，陷宫执念成灾）
