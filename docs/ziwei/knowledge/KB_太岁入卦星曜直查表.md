# 太岁入卦星曜直查表

版本：V1.0

说明：系统拿到入卦者生年后，按以下四张表顺序查询，直接得出所有入卦星曜落位，无需推算。

---

## 查询流程

```
Step 1：确认生年天干 + 生年地支
        ↓
Step 2：查【表一】→ 生年四化 + 禄存 + 擎羊 + 陀罗 + 天魁 + 天钺
        ↓
Step 3：查【表二】→ 红鸾 + 天喜
        ↓
Step 4：查【表三五虎遁干矩阵】→ 得出遁干
        ↓
Step 5：查【表四四化索引】→ 遁干四化
        ↓
Step 6：将以上所有星曜落入命主原局命盘对应宫位，虚拟命盘构建完成
```

## 注意事项

- 四化落位以该星在**命主原局命盘**所在宫位为准，不是以入卦者自身命盘为准
- 生年四化主早（表层/初期心态），遁干四化主晚（深层/后期心态），两套均须完整入卦
- 禄存永远被擎羊（前一位）和陀罗（后一位）夹制，三星须同时入卦
- 天魁天钺须成对入卦，红鸾天喜须成对入卦
