**核心原则：宫位旺弱决定一切**
全系统底层逻辑：宫位旺弱决定该宫所有星曜的呈现面貌与能量强弱。

**极旺/旺宫**

**平宫**

**陷宫/极弱**

**空宫**

空宫一切依赖对宫三合夹宫的投射质量
空间投射亮度原则：
对宫/三合宫/夹宫的星曜投射到本宫时，星曜亮度由该星曜所在宫的旺弱决定（不是本宫旺弱），再乘以空间衰减系数。
例：对宫旺宫有火星 → 火星在旺宫呈正面发挥（被制） → 按旺宫煞星分值 × 0.8 投射本宫，煞力相对轻；对宫陷宫有火星 → 火星逞凶 → 按陷宫煞星分值 × 0.8 投射，煞力较重。

六步评分流程
严格按顺序执行，不得跳步。加分阶段完成后才判断宫位旺弱，再执行减分。
#### 第一步：骨架基础分
从骨架映射库读取本宫主星状态，锁定初始基础分与天花板。Current_Score = 初始基础分。
同时记录主星制煞能力等级：强制煞（紫微/天府）、中制煞（太阳旺/武曲旺/廉贞旺/七杀旺/破军旺/贪狼旺）、弱制煞（天机/太阴/天同/天梁）。
#### 第二步：加分阶段（顺序执行①～⑦）
① 三方四正吉星加分
扫描本宫+对宫+两个三合宫+两侧夹宫，凡见以下吉星，按空间衰减系数加分：
化禄、左辅、右弼、文昌、文曲、天魁、天钺
化权/化科：只记录位置，不加分，仅供格局判断使用
② 命主太岁宫遁干化禄加分
命主生年地支 = 太岁宫，查五虎遁表得遁干，再查遁干四化，化禄落入哪宫则对该宫加分。
③ 父亲生年四化——化禄加分
父亲生年干四化中，化禄落入哪宫则对该宫加分。化权/化科只记录，不加分。
④ 父亲太岁宫遁干化禄加分
父亲生年地支 = 太岁宫，查五虎遁表得遁干，化禄落入哪宫加分。
⑤ 母亲生年四化——化禄加分
母亲生年干四化中，化禄落入哪宫则对该宫加分。化权/化科只记录，不加分。
⑥ 母亲太岁宫遁干化禄加分
母亲生年地支 = 太岁宫，查五虎遁表得遁干，化禄落入哪宫加分。
⑦ 吉格加分
查格局库，成格条件与引动条件均满足则触发，按级别应用倍率（大吉×1.5，中吉×1.3，小吉×1.0只标注）。
三奇嘉会格触发条件：同一来源（命造/父亲/母亲三者之一）的化禄+化权+化科同时会于三方四正，或命宫+两侧夹宫凑齐同一来源的科权禄，则成格（中吉×1.3）。四化来源必须严格隔离，不同来源的科权禄不得混算。
