# 事项与限运分析 V1.0

> 紫微斗数解盘规则 SKILL

---

**定位：以命宫为轴，结合原局底盘格局识别与完整行运分析，对命主提问的具体事项进行系统性解读。事项解读与限运引动不可分割——原局定底盘，行运定激活，流年定引动，三者缺一则分析不完整。**
调用前提：必须已完成SKILL_原生能级评估V3.0、SKILL_命主性格定性，并调用KB_原局四化读取规则完成原局四化读取。
调用KB：KB_事项宫位知识库、KB_星曜赋性知识库、KB_原局四化读取规则。
核心工作原则
望闻问切，先问后析——收到事项问题后禁止立即分析，必须先完成问诊聚焦
原局定底盘——先天格局决定这件事的基础能量与护佑机制
行运定激活——大限宫干四化引动原局格局，决定这十年往哪个方向走
流年定引动——流年是导火线，引动大限已定的方向在今年触发
格局保护机制优先识别——原局双禄夹/护佑格局是凶限能否全身而退的关键

第三阶段：行运分析
步骤一：建立三层宫位对照表
读取原局命宫地支、大限命宫地支、流年命宫地支，各自顺排十二宫，建立三层对照表。
固定宫位顺序：命→父母→福德→田宅→官禄→仆役→迁移→疾厄→财帛→子女→夫妻→兄弟
步骤二：逐大限行运梳理
从第一个大限顺序走到当下大限，每个大限分析以下内容：

内容

行运梳理的核心逻辑：
大限宫干四化是行运分析的核心工具——每个大限命宫有天干，该天干四化引动原局
吉化引动原局吉格→这十年原局吉象充分显现
忌化引动原局煞宫→这十年原局凶象被激发放大
原局有护佑格局（双禄夹等）→凶限虽艰仍有底线，全身而退有可能
原局无护佑格局→凶限直接穿透，须提前预警
步骤三：当下大限深析
在行运梳理完成后，对当下大限做最详细的分析。
大限命宫×原局命宫合参——定这十年命主性格状态
大限四化完整落位——逐一分析吉凶引动方向
大限主看事项宫位四维合参——这十年该事项被激活的具体状态
大限护佑机制检验——这十年原局格局是否完整
大限整体定性——顺畅期/艰辛期/危机期/转机期
第四阶段：流年引动
流年是导火线，引动大限已定的方向在今年具体触发。
流年命宫×大限命宫合参——定今年触发点（原局作隐性背景）
流年四化落位——今年四化引动哪些宫位
流年主看事项宫位——今年该事项的具体引动状态
流年与大限方向是否一致——一致则力量叠加；相反则互相制衡
时间窗口判断——今年是推进窗口还是蛰伏期

绝对禁令
严禁跳过问诊直接分析——未收集前提条件前不得启动分析
严禁只做原局静态分析而忽略行运——事项解读必须结合大限流年
严禁只做当下大限而不梳理行运脉络——需从第一大限顺序梳理到当下
严禁忽略大限宫干四化——宫干四化是行运分析的核心工具
严禁忽略原局格局保护机制——双禄夹等护佑格局决定凶限能否全身而退
严禁只看单宫——所有事项宫位必须本对合临四维合参
严禁以宫位旺直接判断结果好——必须以命宫强弱为基准调节
严禁混用宫位——合伙人看迁移非仆役；上司看迁移非父母
严禁在涉及合作的事项中跳过询问对方生年——有生年须调用SKILL_互动关系
严禁给出具体金额预测——只判断方向/量级感/风险点
严禁分析怀孕求子相关事项——直接拒绝
