# 命主性格定性 V1.4

> 紫微斗数解盘规则 SKILL

---

**定位：以命宫/身宫/太岁宫三宫为核心，结合本对合临四维合参，定性命主完整性格图谱与行为模式。输出结果作为所有后续模块（事项解读、限运引动）的人格底座。**
**边界：本Skill做原局静态性格定性。大限/流年命宫的动态叠加逻辑见「命宫全息底色三层叠加」章节，限运引动的完整分析属于SKILL_限运引动模块。**
**核心原则一：性格决定行为，行为决定结果**
所有事项分析必须先定性命主是什么样的人，再讨论他/她会如何应对，最终才能判断结果走向。禁止跳过性格定性直接分析事项宫位。
**核心原则二：本对合临四维合参**
判断任何一个宫位（包括命宫/身宫/太岁宫），必须同时看四个维度，不能只看单宫。

夹宫成立条件与两侧力量对比规则：
夹宫成立条件：本宫左侧宫＋右侧宫各自有星曜，才构成夹；仅一侧有星不构成夹
严禁将同宫的成对星曜（如昌曲同宫、左右同宫）称为「夹宫」——夹宫须左右两侧各自有星才成立
两侧均衡之夹：左右宫能量相近→本宫稳定，吉夹则安，煞夹则制
两侧悬殊之夹：一侧强旺一侧弱薄→形成不对称夹，须标注落差影响
不对称夹的落差影响：命主在该宫层面感知的资源/支撑来自强侧，弱侧支撑有限；强侧若为煞星或强势星（如紫微权/破军）则易形成强侧主导、弱侧失衡的局面
四维合参的判断顺序：
先看本宫主星旺弱与星性——定性格底色
再看对宫投射——是加强还是制约本宫
看三合宫支撑——三合吉旺则本宫有强力后援，三合有煞则本宫侧翼受压
看夹宫影响——先确认夹宫成立条件，再评估两侧力量是否均衡，悬殊时标注落差方向
综合四维得出本宫综合性格定性
**核心原则三：命宫全息底色——全程带入所有事项分析**
命宫是所有事项分析的统领轴心。命宫内的所有信息——四化（禄权科忌）、六吉星、六煞星、丙丁级星曜——在讨论任何事项时都必须一并参考，不能只提化忌。
