【合参规则（必须遵守）】

- 所有合参结论必须基于数据中的等级对比（吉旺>平>凶弱），不可凭空推断。

【分析规则】
- 等级映射：吉旺（得分≥6.0），平（4.5-5.9），凶弱（<4.5）。
- 十年定性矩阵（原局事项等级 vs 大限事项等级）：
  | 原局 | 大限 | 趋势 |
  |------|------|------|
  | 吉旺 | 吉旺 | 大旺 |
  | 吉旺 | 平   | 小旺 |
  | 吉旺 | 凶弱 | 小衰 |
  | 凶弱 | 吉旺 | 小旺（借运） |
  | 凶弱 | 平   | 小衰 |
  | 凶弱 | 凶弱 | 大衰 |
- 四化引动效果：
  - 禄：结果、财源、顺畅度
  - 权：增强控制力、执行力、企图心、机会
  - 科：增强名声、贵人、条理
  - 忌：增强阻碍、执念、风险
  - 双夹引动效果加倍。
- 流年方向判断（大限吉凶 + 流年吉凶）：
  - 大限吉 + 流年吉 → 最佳推进窗口
  - 大限吉 + 流年凶 → 部分化解（谨慎推进）
  - 大限凶 + 流年吉 → 有波折干扰（维持为主）
  - 大限凶 + 流年凶 → 风险最高（预警规避）
  其中，“大限吉凶”根据大限事项宫位的等级（吉旺为吉，平/凶弱为凶）和四化引动综合判断；“流年吉凶”同理。
- 命宫强旺（吉旺）时，能压制其他宫位的煞星负面影响；命宫偏弱时，易被煞星拖累。

【空宫处理规则】
- 事项宫位空宫 → 借对宫星曜论，吉象减半，凶象更凶；意识形态薄弱，需外力推动。
- 空宫命宫 → 对宫投射为主，须看对宫三合构成何种大格局。
- 空宫不等于没结果，只是意识形态薄弱。
